## 1、申请开通服务商小程序(即JSAPI)支付产品权限

使用服务商模式小程序(即JSAPI)支付产品，涉及两个权限：

- 服务商自身申请开通小程序(即JSAPI)支付权限，[详见1.1](https://pay.weixin.qq.com/doc/v3/partner/4012076731.md#1.1%E3%80%81%E6%9C%8D%E5%8A%A1%E5%95%86%E8%87%AA%E8%BA%AB%E7%94%B3%E8%AF%B7%E5%BC%80%E9%80%9A%E5%B0%8F%E7%A8%8B%E5%BA%8F(%E5%8D%B3JSAPI)%E6%94%AF%E4%BB%98%E6%9D%83%E9%99%90)。

- 子商户（也叫特约商户）授权服务商使用小程序(即JSAPI)支付权限，[详见1.2](https://pay.weixin.qq.com/doc/v3/partner/4012076731.md#1.2%E3%80%81%E5%AD%90%E5%95%86%E6%88%B7%E6%8E%88%E6%9D%83%E6%9C%8D%E5%8A%A1%E5%95%86%E4%BD%BF%E7%94%A8%E5%B0%8F%E7%A8%8B%E5%BA%8F(%E5%8D%B3JSAPI)%E6%94%AF%E4%BB%98%E6%9D%83%E9%99%90)。


### 1.1、服务商自身申请开通小程序(即JSAPI)支付权限

若商户已拥有服务商商户号(如无，请按[服务商商户入驻详细流程](https://kf.qq.com/faq/161220iqeAfA1612202yeURB.html)指引申请)，请按照如下步骤申请，申请完成后进入审核阶段，待审核通过后(7个工作日内)，则开通小程序(即JSAPI)支付权限成功。

|     |     |
| --- | --- |
| 步骤一：登录服务商平台，点击“产品中心- 特约商户授权产品-服务商小程序(即JSAPI)支付” | 步骤二：点击“申请开通” |

### 1.2、子商户授权服务商使用小程序(即JSAPI)支付权限

子商户（子商户需要服务商协助入驻，入驻有两种方式：[服务商平台入驻](https://kf.qq.com/faq/200114EZfmy62001142YVRzy.html)或[接口入驻](https://pay.weixin.qq.com/doc/v3/partner/4012062365.md)）授权服务商使用小程序(即JSAPI)支付权限，涉及两个步骤：

步骤一：服务商登录[服务商平台](https://pay.weixin.qq.com/index.php/partner/public/home)，通过路径【产品中心 -> 特约商户授权产品 -\> 服务商小程序(即JSAPI)支付 -> 特约商户列表】找到需要授权小程序(即JSAPI)支付的子商户号，点击“发起邀请”。

步骤二：子商户登录[商户平台](https://pay.weixin.qq.com/)，通过路径【产品中心 -\> 我的授权产品】找到服务商小程序(即JSAPI)支付，点击“授权”。

| 步骤一：服务商申请授权 | 步骤二：子商户确认授权 |
