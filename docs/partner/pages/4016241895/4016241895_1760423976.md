可参考以下几点排查：

1、检查 HTTP 头 Wechatpay-Serial 的内容是否跟商户当前所使用的微信支付平台证书的序列号或微信支付公钥id一致。

2、按照[如何使用平台证书验签名](https://pay.weixin.qq.com/doc/v3/merchant/4013053420.md)或[如何使用微信支付公钥验签](https://pay.weixin.qq.com/doc/v3/merchant/4013053249.md)指引核对私钥、证书及签名算法（SHA256withRSA），确保验签逻辑与微信支付规则一致。

3、检查 HTTP 头 Wechatpay-Signature 的内容是否以WECHATPAY/SIGNTEST/ 开头，此前缀为签名探测流量，参考[应对签名探测流量](https://pay.weixin.qq.com/doc/v3/merchant/4013053420.md#5.-%E5%BA%94%E5%AF%B9%E7%AD%BE%E5%90%8D%E6%8E%A2%E6%B5%8B%E6%B5%81%E9%87%8F)指引处理。

4、可以使用验签工具验证签名，参考[如何使用签名/验签工具](https://pay.weixin.qq.com/doc/v3/merchant/4012365352.md)。

5、建议使用微信支付官方提供的[SDK](https://pay.weixin.qq.com/doc/v3/merchant/4012076498.md)，SDK封装了签名验签方法。
