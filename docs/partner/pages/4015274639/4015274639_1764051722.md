## 1、开发前准备

### 1.1、设置安全联系人

微信支付日常安全监测发现技术异常时，会向安全联系人和超级管理员发送风险提醒。请商户超级管理员尽快设置技术同事为安全联系人，确保能及时接收异常信息评估业务风险，详见[安全联系人设置指引](https://pay.weixin.qq.com/doc/v3/partner/4012083124.md)。

### 1.2、熟悉微信支付接口规则

正式进入开发前，开发者需要先阅读[基本规则](https://pay.weixin.qq.com/doc/v3/partner/4012081726.md)、[开发参数申请和配置](https://pay.weixin.qq.com/doc/v3/partner/4012081990.md)了解调用微信支付接口的基本规则和签名规则。

### 1.3、准备开发参数

在发起接口请求时，开发者还需传入一些必要参数，如服务商商户号、子商户号（也叫特约商户号）、sp\_appid、证书私钥、公钥等，获取方式详见：[服务商模式开发必要参数说明](https://pay.weixin.qq.com/doc/v3/partner/4013080340.md)。

注意事项：

开发联调阶段，调用接口后，建议在日志中保留应答的HTTP头 `Request-ID` 值， `Request-ID` 作为[请求的唯一标识](https://pay.weixin.qq.com/doc/v3/partner/4012081726.md#%E8%AF%B7%E6%B1%82%E7%9A%84%E5%94%AF%E4%B8%80%E6%A0%87%E8%AF%86)，在调用接口遇到问题时，可向微信侧提供该值用于快速定位到请求记录，协助排查问题原因。

- 商家名片会员是商家名片的功能之一，在接入前需先[申请开通商家名片](https://pay.weixin.qq.com/doc/v3/partner/4016433970.md)功能，再联系微信支付运营开通商家名片会员的权限，开通后即可创建会员卡模版。

### 3.2、用户入会

#### 3.2.1、【新用户】通过商家小程序/H5入会

用户可通过小程序或H5页面进入微信支付提供的开卡组件发起入会流程，核心流程如下：

包括商户[发起预授权](https://pay.weixin.qq.com/doc/v3/partner/4015336986.md)、[拉起组件](https://pay.weixin.qq.com/doc/v3/partner/4015331476.md)、[接收事件通知](https://pay.weixin.qq.com/doc/v3/partner/4015283692.md)、[同步开通结果](https://pay.weixin.qq.com/doc/v3/partner/4015336979.md)、[查询用户会员卡信息](https://pay.weixin.qq.com/doc/v3/partner/4015336980.md)等操作。

时序图中绿色步骤可参考API文档中对应的接口。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/c21616a41d516a7010bee698c60cd567.png)

#### 3.2.2、【新用户】通过商家名片入会

商户通过[创建会员卡模板](https://pay.weixin.qq.com/doc/v3/partner/4015336970.md)接口创建的会员卡模块如符合条件（卡模板状态为生效、 `valid_date_information` 在有效期内且该用户未参与过入会）在商家名片会展示入会入口。商家可以通过need\_pinned参数置顶某张卡模板。用户通过微信支付的商家名片发起入会流程时，核心流程如下：

包括商户[接收事件通知](https://pay.weixin.qq.com/doc/v3/partner/4015283692.md)、[同步开通结果](https://pay.weixin.qq.com/doc/v3/partner/4015336979.md)等操作。

时序图中绿色步骤可参考API文档中对应的接口。

##### 3.2.2.1 普通会员卡

注意：

普通会员，需完成 “ 会员开通” 闭环。用户在商家名片完成开通，然后由微信支付通知商户系统开通结果，最后由商户系统同步微信支付开通结果后才能结束流程。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/29659bc15ffcff8ad64fe9a54710d278.png)

- 付费会员，该功能暂未上线。

包括[导入用户会员卡](https://pay.weixin.qq.com/doc/v3/partner/4015336981.md)、[修改用户会员卡信息](https://pay.weixin.qq.com/doc/v3/partner/4015336985.md)等操作

时序图中绿色步骤可参考API文档中对应的接口。

注意：

- 通过API导入用户会员卡时，必须传入会员卡code字段（user\_card\_code）。

- 根据手机号导入用户会员卡，该功能暂未上线。


![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/3aee189bc9cca8aab18940caff60ed3e.png)

#### 3.2.4、【存量会员】通过下单同步入会

商家可通过在微信支付下单接口的attach字段传入会员卡code（ `user_card_code`）来导入未开通会员的用户。下单用户若此前未开通会员卡，通过此方式可实时添加会员并自动生效一张会员卡，详细指引请参考：[实时动态传入的方式](https://pay.weixin.qq.com/doc/v3/partner/4015534755.md#2.3%E3%80%81%E5%AE%9E%E6%97%B6%E5%8A%A8%E6%80%81%E4%BC%A0%E5%85%A5%E7%9A%84%E6%96%B9%E5%BC%8F)

注意：

通过下单同步入会时，必须传入会员卡code字段（user\_card\_code）。

### 3.6、积分兑券能力接入步骤

#### 3.6.1 前置准备（创建积分兑券模版）

商家按照如下信息，提供给微信支付运营（后续将支持商家自己配置管理），上架每个品牌ID商家内的积分货架内容。

以下表格内容，每创建一次，将获得一个积分兑券模版ID，对应图示中一个红色圈起的模块。

| 字段 | 备注 | 图示 |
| :-- | :-- | :-- |
| 商家brandID |  | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/0a1cdf070010821a6d1d97707a5bd5e8.png) |
| 会员卡模版ID | 配置到哪张会员卡 |
| 消耗积分 | 兑换这一张券，需要消耗多少积分 |
| 总库存 | 在架有多少份这个「x积分兑换xxx券」的福利 |
| 用户兑换单日限额 | 用户每一天可以兑换多少次<br>每一次用户请求，微信支付都会问询商家是否可兑换。该字段仅作为一个前置拦截规则。 |
| 排序ID | 在积分货架上，当前积分兑券模版排序第几顺位。 |
| 商品券ID | 同品牌ID商家下，已创建好的商品券ID |

#### 3.6.2 商家系统内，潜在要做的准备

1. 商家要维护下单实时动态模版ID （这个ID 所有商户都一样，是微信给出来的），[支持的模版内容详见这里。](https://pay.weixin.qq.com/doc/v3/partner/4015534755.md#2.2%E3%80%81%E5%AE%9E%E6%97%B6%E5%8A%A8%E6%80%81%E6%94%AF%E6%8C%81%E7%9A%84%E5%86%85%E5%AE%B9)

   1. 不同的动态模版ID内容不一样，积分兑券业务可以使用类别为「积分兑券」的模版。
2. 商家支付系统要根据openid查会员可用积分。

3. 商家需要根据当前下单顾客的积分数，去匹配积分兑券相关的动态模版ID拼接到下单接口的attach字段里。（如果商家需要在支付凭证cell展示引导用户积分兑券的内容时需要）

| 查询会员卡模板列表 | 通过调用「[查询会员卡模板列表](https://pay.weixin.qq.com/doc/v3/partner/4015336976.md)」接口可查询指定某品牌的所有会员卡模板列表。 |
| 查询会员卡模板信息 | 通过调用「[查询会员卡模板信息](https://pay.weixin.qq.com/doc/v3/partner/4015336974.md)」接口可查询指定会员卡模板的所有信息。 |
| 修改会员卡模板信息 | 通过调用「[修改会员卡模板信息](https://pay.weixin.qq.com/doc/v3/partner/4015336977.md)」接口可更新会员卡的信息，包括基本信息、储值信息、开卡信息等 |

| 查询用户在品牌下所有会员卡 | 通过调用「[查询用户在品牌下所有会员卡](https://pay.weixin.qq.com/doc/v3/partner/4015336984.md)」接口可查询用户在当前品牌下领取的会员卡列表。 |
| 修改用户会员卡信息 | 通过调用「[修改用户会员卡信息](https://pay.weixin.qq.com/doc/v3/partner/4015336985.md)」接口可更新指定用户的会员卡背景图、会员卡号、等级等信息。 |

| 创建用户动态信息 | 通过调用「[创建用户动态信息](https://pay.weixin.qq.com/doc/v3/partner/4015336987.md)」接口可同步非微信支付渠道的会员动态信息到微信支付商家名片会员。 |
| 图片上传 | 通过调用「[图片上传](https://pay.weixin.qq.com/doc/v3/partner/4015900513.md)」接口上传图片后可获得图片URL地址。图片URL可在微信支付商家名片会员相关的API使用，包括会员卡模板、用户会员卡等。 |
