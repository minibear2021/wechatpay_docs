>更新时间：2026.06.29

对商户提供同步车辆入场通知接口




## 接口说明

支持商户：【普通服务商】　【平台商户】

请求方式：【POST】 `/v3/parking/reminders/entry`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/merchant/4012365342.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `application/json` |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| out_serial_number | string(64) | true | 【商户停车单号】 商户系统自行生成，须在同一商户号下全局唯一。建议不超过32字符，超过64字符将被拦截。 |
| parking_id | string(64) | true | 【微信停车场ID】 停车场进件审核通过且运营入驻完成后，由微信支付分配。可通过「查询停车场进件申请单」或「查询停车场信息」接口获取；入场/离场/支付等接口须与入场时保持一致。 |
| plate_number | string(16) | true | 【车牌号】 国内车牌，省份简称+6~7位字母或数字（新能源绿牌8位）。 |
| enter_timestamp | integer | true | 【车辆入场时间】 商户需要传入一个秒级的时间戳 |
| plate_color | string | true | 【车牌颜色】 按照本文档提供的车牌颜色枚举值传入，各枚举值含义见字段枚举说明。<br>可选取值<br><ul><li>`PLATE_COLOR_UNKNOWN`:  未知</li><li>`PLATE_COLOR_BLUE`:  蓝色</li><li>`PLATE_COLOR_YELLOW`:  黄色</li><li>`PLATE_COLOR_BLACK`:  黑色</li><li>`PLATE_COLOR_WHITE`:  白色</li><li>`PLATE_COLOR_GREEN`:  绿色</li><li>`PLATE_COLOR_YELLOW_GREEN`:  黄绿</li><li>`PLATE_COLOR_AGRICULTURAL_YELLOW`:  农黄</li><li>`PLATE_COLOR_OTHER`:  其他</li></ul> |
| car_type | string | true | 【车辆性质】 按照本文档提供的车辆性质枚举值传入，各枚举值含义见字段枚举说明。<br>可选取值<br><ul><li>`PLATE_TYPE_UNKNOWN`:  无(缺省)</li><li>`PLATE_TYPE_TEMPORARY`:  临停</li><li>`PLATE_TYPE_MONTHLY`:  月卡</li><li>`PLATE_TYPE_FREE`:  免费车辆</li><li>`PLATE_TYPE_PREPAID`:  储值车</li><li>`PLATE_TYPE_OTHERS`:  其他</li></ul> |
| vehicle_type | string | true | 【车辆类型】 按照本文档提供的车辆类型枚举值传入，各枚举值含义见字段枚举说明。<br>可选取值<br><ul><li>`VEHICLE_TYPE_UNKNOWN`:  无(缺省)</li><li>`VEHICLE_TYPE_SMALL`:  小车</li><li>`VEHICLE_TYPE_LARGE`:  大车</li><li>`VEHICLE_TYPE_EXTRA_LARGE`:  超大车</li><li>`VEHICLE_TYPE_ALL_TYPE`:  全车型</li></ul> |
| entrance_number | string(128) | false | 【停车场入口编号】 停车场入口编号，由大小写字母和数字组成，如 A01。长度1~128字符。 |
| entrance_name | string(64) | false | 【停车场入口名称】 停车场入口名称。长度1~64字符。 |
| discount_template_id | array[integer] | false | 【优惠信息模板ID】 单次最多传3个模板ID |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| code | string(32) | false | 【响应码】 仅接口调用失败时返回；调用成功时不返回此字段。失败时返回错误码字符串，具体见接口错误码列表。 |
| message | string(256) | false | 【返回信息】 仅接口调用失败时返回；调用成功时不返回此字段。失败时返回错误原因描述。 |
| serial_number | string(64) | false | 【微信停车单号】 接口调用成功时返回，为微信支付侧停车单唯一标识。 |

| 400 | INVALID\_REQUEST | HTTP 请求不符合微信支付 APIv3 接口规则 | 请参阅 [接口规则](https://pay.weixin.qq.com/doc/v3/merchant/4012081709.md) |
| 401 | SIGN\_ERROR | 验证不通过 | 请参阅 [签名常见问题](https://pay.weixin.qq.com/doc/v3/merchant/4012072670.md) |
