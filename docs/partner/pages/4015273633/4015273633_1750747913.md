服务商通过[品牌入会组件预授权接口](https://pay.weixin.qq.com/doc/v3/partner/4015336986.md)获取授权的必要参数token后，可在小程序内使用微信支付提供的小程序方法调起品牌会员入会组件，引导用户完成入会流程。

如果要在该组件中展示优惠券，促进用户的点击转化行为，需要接入[「创建品牌会员发券活动」](https://pay.weixin.qq.com/doc/v3/partner/4016464918.md)接口创建发券活动，活动创建后将取最新的有效活动设置的优惠券展示在此组件内。 如果不需要在改组件中展示优惠券，可不用调用创建品牌会员发券接口，这样组件上就不会展示优惠券。

## 接口说明

支持商户： 【普通服务商】

兼容性表现说明

- 小程序版本库 >= 2.28.1 为最佳体验；最低版本为 2.6.0，低版本无法打开组件，需提示用户升级微信版本。

- iOS 兼容性表现：若微信版本>=7.0.3，开发者可以通过此 openSDK 接口拉起品牌会员入会组件；若微信版本 < 7.0.3，需提示用户升级微信版本。

- Android 兼容性表现：若微信版本>=7.0.3，开发者可以通过此 openSDK 接口拉起品牌会员入会组件；若微信版本 < 7.0.3，需提示用户升级微信版本。


接口名称: wx.openBusinessView

## 字段说明

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| businessType | string(32) | true | 【跳转类型】固定值，请传入 wxpayCardMember |
| queryString | string(2048) | true | 【业务参数】需要传递给入会组件的数据。使用 URL 的 query string 方式传递参数，格式为key=value&key2=value2，其中 value，value2 需要进行 UrlEncode 处理。 |
| queryString.token | string(32) | true | 【预授权 token】调用品牌预授权接口返回的预授权 token (1 小时内有效)。 |
| queryString.scene | string(32) | true | 【组件场景值】组件的应用场景值，固定为 `CARD_MEMBER` |
| success | Function | false | 【调用成功的回调函数】 接口调用成功的回调函数 |
| fail | Function | false | 【调用失败的回调函数】 接口调用失败的回调函数 |
| complete | Function | false | 【调用结束的回调函数】 接口调用结束的回调函数（调用成功、失败都会执行） |

wx.openBusinessView 请求示例

```
if (wx.openBusinessView) {
  wx.openBusinessView({
    businessType: 'wxpayCardMember',
    queryString: 'token=EhhEF0Jt0uDLPK6Cr70KCrJmkTtfPPE2&scene=CARD_MEMBER',
    success() {
      // dosomething
    },
    fail() {
      // dosomething
    },
    complete(e) {
	  console.log('openBusinessView complete', `回调：${e.extraData}`);
    }
  });
} else {
  // 引导用户升级微信版本
}
```

### 返回参数

触发场景: 用户从品牌会员入会组件页返回到品牌小程序页面。

回调方法:

1. 请求参数中的 complete 回调函数中可获取到数据

2. 品牌小程序可在 App.onShow 中获取到数据，具体参考[框架接口-小程序App](https://developers.weixin.qq.com/miniprogram/dev/reference/api/App.html#onShow-Object-object)


注意：

- 只有用户点击入会组件页面内返回按钮时，才会带上返回参数；如果用户点击小程序页面左上角的返回图标按钮，或左滑退出则不会带上返回参数。

- 前端回调参数无法判断用户是否授权成功，具体状态需以[查询用户在品牌下所有会员卡](https://pay.weixin.qq.com/doc/v3/partner/4015336984.md)和[会员卡事件通知](https://pay.weixin.qq.com/doc/v3/partner/4015283692.md)返回的结果为准。


| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| extraData | Object | true | 【业务参数】回传商家小程序的业务数据 |
| extraData.errcode | number | true | 【错误码】表示入会流程是否成功，非 0 表示有异常 |
| extraData.msg | string(256) | true | 【状态描述】状态描述 |
| extraData.data | Object | false | 【详细信息】详细信息，当errcode为0时才返回；反之，则不返回。 |
| extraData.data.hasJoinCardMember | boolean | true | 【用户是否入会成功】表示用户入会是否成功 |
