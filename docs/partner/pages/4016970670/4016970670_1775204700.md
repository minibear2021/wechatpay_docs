## 1、如何接入亲情医保移动支付

前提：定点机构已经接入或者准备接入本人的医保移动支付。

亲情医保移动支付，由两块组成：授权+支付。因此基于本人医保移动支付已接入的前提，要实现亲情移动支付，需要在这两个地方进行改造。

### 1.1 接入准备

#### 1.1.1 立项

如果定点机构之前已经对接过微信的医保移动支付了，则在原先的立项邮件中回复标题新增对接”国标亲情移动支付“给对应的商务，商务内部立项后，进行国标亲情移动支付对接。

如果是正准备对接本人和亲情医保移动支付，则按照“权限申请——医院立项”部分进行立项申请。

#### 1.1.2 测试用户添加

测试亲情移动支付，本人需要添加到微信及医保局测试环境（见“[开发前准备](https://pay.weixin.qq.com/doc/v3/partner/4017149893.md)”），同时，需要测试的亲人也在医保局测试环境中（见“[开发前准备](https://pay.weixin.qq.com/doc/v3/partner/4017149893.md)”），需要把测试的亲人和本人一齐添加到微信测试库。（建议测试环境验证时，使用没有领取过医保电子凭证的亲人，例如16岁以下的小孩或者老人，用于测试）。

【微信亲情账户测试环境信息添加模板】（仅限测试环境调试使用）

| 本人姓名： | 亲人姓名： |
| :-- | :-- |
| 本人身份证： | 亲人身份证： |
| 本人手机号： |  |
| 本人微信号： |  |

如果定点机构是要在正式环境调试的话，添加亲属路径为：微信小程序“我的医保凭证”-刷医保码-亲情账户，去进行添加和激活后使用。或者是在授权过程中添加和激活。

### 1.2 接入调试

#### 1.2.1 授权接口

基于之前的本人的授权链接，亲情授权需要在授权链接中拼接familyid，例如是帮亲属A完成亲属医保移动支付，则familyid则是亲属A的“姓名+身份证后4位”的md5，即指定帮这个亲属A进行亲情支付的授权，获取到亲情授权码。具体的拼接方式见[用户授权接入文档](https://docs.qq.com/doc/DV3JYRG1xelhKTWdz)。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/80ba963cba971aeeaea15d05bed80e71.jpeg)

#### 1.2.2 下单接口

亲情移动支付是由大人B，帮亲属A完成线上的医保支付，所以在医保自费混合收款下单接口的relative字段中，需要传入的是亲属A的实名信息（即就诊人），而在payer字段传入大人B（即本人）的实名信息，并且在支付授权码字段传入的是亲情授权码，从而实现医保移动支付。具体的亲属扩展字段传入见[医保自费混合收款下单](https://pay.weixin.qq.com/doc/v3/partner/4012503131.md)。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/d9b00edfce4e8ea78ea6facb97bccbd1.jpeg)

中台会校验授权时生成的亲情授权码和支付接口的亲情授权码，是否在中台有对应的亲属绑定记录，如果没有，则支付会报错。

#### 1.2.3 费用明细上传

医院和中台的接口没有变化：传的就诊人就是就医人A实名信息和就医人A诊断信息。

#### 1.2.4 如何测试

- 本人的医保移动支付已线上运行


如果本人的医保移动支付已经在线上运行了，要验证的话，有两条路径：使用沙箱环境，直接正式环境验证。

使用沙箱环境：测试人和测试的亲属，两个人的实名信息，加入到微信测试环境；并加入到医保局测试环境。条件后，重新激活本人及代激活亲属，实现亲情授权和支付的测试。（暂只支持该方式调试）

直接正式环境验证：通过正式的微信亲情授权链接进行验证测试，可提前添加亲属并代激活，也可以在授权过程中，进行亲情的添加及代激活，然后完成亲情支付。

- 本人的医保移动支付还在测试环境调试


按照测试环境流程，调试本人和亲情的医保移动支付。调试完成后，进行切换申请。

#### 1.2.5 如何上线

- 本人的医保移动支付已线上运行


不管是按照沙箱环境还是正式环境调试，调试完成后，提供对应的亲情医保移动支付的录屏（含[“UI规范示例”](https://pay.weixin.qq.com/doc/v3/partner/4016824698.md#4%E3%80%81UI%E8%A7%84%E8%8C%83%E4%B8%8E%E9%A1%B5%E9%9D%A2%E8%AF%B4%E6%98%8E)相关页面）给微信侧联调同学配置后，可切换线上使用。

- 本人的医保移动支付还在测试环境调试


按照“[验收及环境切换](https://pay.weixin.qq.com/doc/v3/partner/4016970677.md#1.%E7%A7%BB%E5%8A%A8%E6%94%AF%E4%BB%98%E9%AA%8C%E6%94%B6%E5%BD%95%E5%B1%8F%E8%AF%B4%E6%98%8E%EF%BC%88%E9%87%8D%E8%A6%81%EF%BC%8C%E8%AF%B7%E5%8C%BB%E9%99%A2%E7%9B%B8%E5%85%B3%E4%BA%BA%E5%91%98%E4%B8%80%E5%AE%9A%E6%9F%A5%E7%9C%8B%EF%BC%89)”进行测试环境切换，给微信侧联调同学提供对应的本人和亲情医保移动支付录屏，等待处理后切换线上使用。
