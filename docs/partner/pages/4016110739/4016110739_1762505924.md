开发者可通过调用wx.openBusinessView方式在商户小程序打开摇优惠核销组件，引导用户核销商品券。

### 版本更新说明

| 日期 | 改动说明 |
| :-- | :-- |
| 2025-04-22 | 新增全场券点击事件selectGlobalCoupon |

## 接口说明

支持商户： 【普通服务商】

接口名称：wx.openBusinessView

接口兼容：微信 >= 8.0.30&小程序版本库 >= 2.28.1，低版本需提示用户升级到最新的微信版本。

## 字段说明

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| businessType | string | true | 固定配置：wxpayCouponUse |
| extraData | Object | true | 用于向组件传递商品券信息 |
| extraData.couponCode | string | true | 【券的唯一标识】商品详情页跳转商家小程序的路径query会增加该属性，开发者可通过小程序页面query参数获取此值。 |
| extraData.stockId | string | true | 【批次号】商品详情页跳转商家小程序的路径query会增加该属性，开发者可通过小程序页面query参数获取此值。 |
| extraData.authState | string | true | 【用户登录账号的状态】<br>`verified`：用户正确登录账号<br>`invalidCredentials`：用户登录错误账号<br>`unauthenticated`：用户未登录/注册<br>具体可参考文档 [开发指引](https://pay.weixin.qq.com/doc/v3/partner/4016110741.md) |

wx.openBusinessView 请求示例

```
if (wx.openBusinessView && checkVersion()) {
//checkVersion版本检查：微信>=8.0.30&小程序版本库 >= 2.28.1
  wx.openBusinessView({
    businessType: 'wxpayCouponUse',
    extraData: {
      couponCode: '1227944959000000911017', // 从query获取
      stockId: '1286950000000039',   // 从query获取
      authState: 'verified' // 见本文档摇优惠-核销组件（通过openid发券）开发指引 - step2、判断用户登录账号状态
    },
    success(e) {
        console.log(e.extraData.action); // 用户动作
        console.log(e.extraData.productPath); // 用户选择的商品路径
      // dosomething
    },
    fail() {
      // dosomething
    },
    complete() {
      // dosomething
    }
  });
} else {
  // 引导用户升级微信版本
}
```

### 返回参数

success回调结果参数

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| extraData | Object | true | 用于向组件传递商品券信息 |
| extraData.action | string | false | 【用户动作】<br>`selectProduct`: 选择商品<br>`selectGlobalCoupon`：选择全场券<br>`auth`: 注册/登录<br>`closePage`: 关闭页面<br>`switchAccount`: 更换登录账号 |
| extraData.productPath | string | false | 【用户选择的商品路径】当用户选择商品，会在success回调返回商品路径，由开发者自行跳转 |
| extraData.errMsg | string | false | 【错误信息】 |

### 小程序生命周期说明

1. 商户小程序打开摇优惠核券组件时，会触发 [App onShow](https://developers.weixin.qq.com/miniprogram/dev/reference/api/App.html#onShow-Object-object) 和 [Page onShow](https://developers.weixin.qq.com/miniprogram/dev/reference/api/Page.html#onShow) 事件

2. 当摇优惠核券组件关闭时，会触发 [App onHide](https://developers.weixin.qq.com/miniprogram/dev/reference/api/App.html#onHide) 和 [Page onHide](https://developers.weixin.qq.com/miniprogram/dev/reference/api/Page.html#onHide) 事件
