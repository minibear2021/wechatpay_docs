从业机构调用该接口向微信医保后台通知医保订单的退款成功结果




## 接口说明

支持商户：【从业机构（银行）】　【普通服务商】　【从业机构（支付机构）】

请求方式：【POST】 `/v3/med-ins/refunds/notify`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| sub_mchid | string(32) | true | 【医疗机构的商户号】 医疗机构的商户号 |
| med_refund_total_fee | integer | true | 【医保退款的总金额】 单位分，医保退款的总金额。 |
| med_refund_gov_fee | integer | true | 【医保统筹退款金额】 单位分，医保统筹退款金额。 |
| med_refund_self_fee | integer | true | 【医保个账退款金额】 单位分，医保个账退款金额。 |
| med_refund_other_fee | integer | true | 【医保其他退款金额】 单位分，医保其他退款金额。 |
| refund_time | string(64) | true | 【医保退款成功时间】 遵循rfc3339标准格式，格式为yyyy-MM-DDTHH:mm:ss+TIMEZONE。 |
| out_refund_no | string(64) | true | 【从业机构\\服务商退款单号】 有自费单时，从业机构\\服务商应填与 [自费退款申请](https://pay.weixin.qq.com/doc/v3/merchant/4013071036.md) 处一致的out\_refund\_no。否则从业机构透传医疗机构退款单号即可。 |
