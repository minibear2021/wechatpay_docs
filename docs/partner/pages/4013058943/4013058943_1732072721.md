## 1、什么时候需要更换？

（1）商户API证书丢失、泄露需要及时更换

（2）商户API证书过期

## 2、如何更换商户API证书？

### 第一步：申请一张新的商户API证书

参考[申请商户API证书](https://pay.weixin.qq.com/doc/v3/merchant/4012072428.md)

### 第二步：替换商户API证书和证书序列号

- APIv3接口代码中，【商户API证书私钥】和【商户API证书序列号】需替换为新证书的值。

- APlv2接口代码中，需要替换请求接口时携带的apiclient\_cert.p12商户API证书文件，详细参考[商户API证书](https://pay.weixin.qq.com/doc/v2/merchant/4011985891.md#4%E3%80%81API%E8%AF%81%E4%B9%A6)。


### 第三步：作废旧商户API证书

替换完证书和序列号后，建议先[观察几天旧证书昨日调用量](https://pay.weixin.qq.com/doc/v3/merchant/4012072428.md#Q%EF%BC%9A%E5%A6%82%E4%BD%95%E6%9F%A5%E7%9C%8B%E5%95%86%E6%88%B7API%E8%AF%81%E4%B9%A6%E4%BD%BF%E7%94%A8%E9%87%8F%EF%BC%9F)是否为0，确认无误后再作废旧商户API证书。作废7天内如有异常可以取消作废，暂时[恢复旧证书](https://pay.weixin.qq.com/doc/_iwiki_staging/4013042002.md#3%E3%80%81%E5%95%86%E6%88%B7API%E8%AF%81%E4%B9%A6%E4%BD%9C%E5%BA%9F%E5%90%8E%E5%A6%82%E4%BD%95%E6%81%A2%E5%A4%8D%EF%BC%9F)。

商户号超级管理员登录[微信商户平台](https://pay.weixin.qq.com/)，在【账户中心 \- 账户设置 \- API安全 - 管理证书】作废证书。以下为具体操作步骤：

|     |     |
| --- | --- |
| 步骤一：登录商户平台，点击“账户中心-API安全-管理证书” | 步骤二：在API证书管理页面，点击对应证书旁边的“作废”按钮 |

## 3、商户API证书作废后如何恢复？

商户API证书自作废之日起7日内仍可恢复使用，商户号超级管理员登录[微信商户平台](https://pay.weixin.qq.com/)，在【账户中心 \- 账户设置 \- API安全 - 管理证书】恢复证书。以下为具体操作步骤：

|     |     |
| --- | --- |
| 步骤一：登录商户平台，点击“账户中心-API安全-管理证书” | 步骤二：在API证书管理页面，点击对应证书旁边的“恢复”按钮 |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/fdf561b486175822c7cd541f21f77bab.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/a7aa3ce4371efee6b4801ff4116f82ee.png) |
| 步骤三：在页面弹框中点击“确认恢复” | 步骤四：安全验证，输入操作密码，短信验证码，点击“确定” |
