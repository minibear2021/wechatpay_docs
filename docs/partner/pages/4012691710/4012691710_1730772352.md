电商平台服务商调用注销申请接口时，需要先调用本接口上传相关的资料图片，获取图片ID后，再填写到注销申请请求中。




## 接口说明

支持商户：【普通服务商】【平台商户】

请求方式：【POST】 `/v3/ecommerce/account/cancel-applications/media`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/partner/4012082567.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/partner/4012365870.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `multipart/form-data` |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| file | message | true | 将媒体图片进行二进制转换，得到的媒体图片二进制内容，在请求body中上传此二进制内容。媒体图片只支持JPG、BMP、PNG格式，文件大小不能超过2M。 |
| meta | object | true | 媒体文件元信息 |
| meta.file_name | string(128) | true | **【文件名称】** 商户上传的媒体图片的名称，商户自定义，必须以JPG、BMP、PNG为后缀。 |
| meta.file_digest | string(64) | true | **【文件摘要】** 图片文件的文件摘要，即对图片文件的二进制内容进行sha256计算得到的值，大小写均可。 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| media_id | string(512) | true | **【媒体文件标识ID】** 微信返回的媒体文件标识ID |
## 错误码

### 公共错误码

| 状态码 | 错误码 | 描述 | 解决方案 |
| --- | --- | --- | --- |
| 400 | PARAM\_ERROR | 参数错误 | 请根据错误提示正确传入参数 |
| 400 | INVALID\_REQUEST | HTTP 请求不符合微信支付 APIv3 接口规则 | 请参阅 [接口规则](https://pay.weixin.qq.com/doc/v3/partner/4012081726.md) |
| 401 | SIGN\_ERROR | 验证不通过 | 请参阅 [签名常见问题](https://pay.weixin.qq.com/doc/v3/partner/4012082324.md) |
| 500 | SYSTEM\_ERROR | 系统异常，请稍后重试 | 请稍后重试 |

### 业务错误码

| 状态码 | 错误码 | 描述 | 解决方案 |
| --- | --- | --- | --- |
| 500 | SYSTEM\_ERROR | 系统错误 | 系统异常，请使用相同参数稍后重新调用 |
| 400 | INVALID\_REQUEST | 图片内容过大 | 图片内容大小超出限制，请使用2M以内的图片 |
| 400 | INVALID\_REQUEST | 图片Content-Type非法 | 请更换为JPG、JPEG、BMP、PNG格式的图片 |
