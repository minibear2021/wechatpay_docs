图片上传

更新时间：2024.11.05

电商平台服务商调用注销申请接口时，需要先调用本接口上传相关的资料图片，获取图片ID后，再填写到注销申请请求中。

接口说明

支持商户：【普通服务商】【平台商户】

请求方式：【POST】/v3/ecommerce/account/cancel-applications/media

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引点击查看

请求参数
折叠全部参数

HeaderHTTP头参数

Authorization 必填　string

请参考签名认证生成认证信息

Accept 必填　string

请设置为application/json

Content-Type 必填　string

请设置为multipart/form-data

BODY包体参数

file 必填 message

将媒体图片进行二进制转换，得到的媒体图片二进制内容，在请求body中上传此二进制内容。媒体图片只支持JPG、BMP、PNG格式，文件大小不能超过2M。

meta 必填 object

媒体文件元信息

属性

file_name 必填 string(128)

【文件名称】商户上传的媒体图片的名称，商户自定义，必须以JPG、BMP、PNG为后缀。

file_digest 必填 string(64)

【文件摘要】图片文件的文件摘要，即对图片文件的二进制内容进行sha256计算得到的值，大小写均可。

请求示例

POST

1String uploadPath = "上传地址";
2String imagePath = "图片地址";
3String sha256 = "sha256数据";
4String meta = String.format("{\"file_name\":\"%s\",\"file_digest\":\"%s\"}", file_name, sha256);        FileUploadResponse fileUploadResponse = fileUploadService.uploadImage(uploadPath, meta, imagePath);

JAVA详细请求示例见：wechatpay-java

应答参数

200 OK

media_id 必填 string(512)

【媒体文件标识ID】微信返回的媒体文件标识ID
