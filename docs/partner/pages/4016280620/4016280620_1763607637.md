服务商可以通过该接口将品牌的门店列表批量与商品券批次组内所有批次关联

前置条件：

1. 已创建 `usage_mode` 为 `PROGRESSIVE_BUNDLE` 的商品券对应的批次组

2. 批次组内批次的 `store_scope` 为 `SPECIFIC`

3. 批次组内批次未失效或过期





## 接口说明

支持商户：【普通服务商】

请求方式：【POST】 `/v3/marketing/partner/product-coupon/product-coupons/{product_coupon_id}/stock-bundles/{stock_bundle_id}/associate-stores`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

接口限频：20/秒（商户号维度）

### Path 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| product_coupon_id | string | true | 【商品券ID】 商品券的唯一标识， [创建商品券](https://pay.weixin.qq.com/doc/v3/partner/4016434630.md) 时由微信支付生成 |
| stock_bundle_id | string | true | 【批次组ID】 商品券批次组的唯一标识，【 [创建商品券(多次优惠)](https://pay.weixin.qq.com/doc/v3/partner/4016434630.md)】或【 [添加商品券批次组](https://pay.weixin.qq.com/doc/v3/partner/4016280622.md)】时由微信支付生成，本接口会同步关联门店信息到批次组内所有批次。请确保该批次组属于 `product_coupon_id` 对应的商品券，且 `store_scope` 为 `SPECIFIC`，否则无法关联门店 |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| store_list | array[object] | true | 【门店列表】 商户欲关联的门店列表，<br>单次最多可批量关联200个门店 |
| store_list.store_id | string | true | 【门店ID】 门店的唯一标识，调用【 [创建品牌门店API](https://pay.weixin.qq.com/doc/v3/partner/4015783183.md)】时由微信支付生成 |
| brand_id | string | true | 【品牌ID】 微信支付为品牌方分配的唯一标识，该品牌应与服务商存在授权关系 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| total_count | integer | true | 【去重后门店总数】 系统会自动对请求中的门店去重，这里是去重后的门店总数 |
| success_store_list | array[object] | false | 【关联成功的门店列表】 关联成功的门店列表 |
| success_store_list.store_id | string | true | 【门店ID】 门店的唯一标识，调用【 [创建品牌门店API](https://pay.weixin.qq.com/doc/v3/partner/4015783183.md)】时由微信支付生成 |
| failed_store_list | array[object] | false | 【关联失败的门店列表】 每个关联失败门店的详细错误信息 |
| failed_store_list.store_id | string | true | 【门店ID】 关联失败的门店ID |
| failed_store_list.code | string(40) | true | 【失败错误码】 门店关联失败的错误码 |
| failed_store_list.message | string(200) | true | 【失败错误信息】 门店关联失败的错误描述 |
## 错误码

以下是本接口返回的错误码列表。详细错误码规则，请参考[微信支付接口规则-错误码和错误提示](https://pay.weixin.qq.com/doc/v3/partner/4012081726.md#%E9%94%99%E8%AF%AF%E7%A0%81%E5%92%8C%E9%94%99%E8%AF%AF%E6%8F%90%E7%A4%BA)

| 状态码 | 错误码 | 描述 | 解决方案 |
| --- | --- | --- | --- |
| 400 | PARAM\_ERROR | 参数错误 | 请根据错误提示正确传入参数 |
| 400 | INVALID\_REQUEST | HTTP 请求不符合微信支付 APIv3 接口规则 | 请参阅 [接口规则](https://pay.weixin.qq.com/doc/v3/merchant/4012081709.md) |

| 500 | SYSTEM\_ERROR | 系统异常，请稍后重试 | 请稍后重试 |
| 400 | INVALID\_REQUEST | 已过期的批次不可关联门店 | 请确认批次是否处于有效期 |
| 400 | INVALID\_REQUEST | 只有 store\_scope 为 SPECIFIC 的批次可以关联门店 | 请确认批次 store\_scope 为 SPECIFIC 再调用本接口 |
| 404 | NOT\_FOUND | 未找到 product\_coupon\_id 对应的商品券 | 请确认 product\_coupon\_id 存在且属于当前品牌 |
| 404 | NOT\_FOUND | 未找到 stock\_id 对应的商品券批次 | 请确认 stock\_id 存在且属于当前商品券 |
| 429 | RATELIMIT\_EXCEEDED | 请求超过接口频率限制 | 请稍后使用原参数重试 |
