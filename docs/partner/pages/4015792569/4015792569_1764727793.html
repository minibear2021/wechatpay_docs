下载发票文件

更新时间：2025.12.03

调用【获取发票下载信息】接口后，调用该接口下载发票文件。该接口无需签名和验签

接口说明

适用对象： 普通服务商

请求URL： 通过【获取发票下载信息】接口，获取“download_url”参数作为请求URL

请求方式： GET

前置条件： 通过【获取发票下载信息】接口获取到“download_url”参数作为请求URL，该URL有效期为30秒，仅可用于下载状态为“ISSUED”的发票

字段说明

请求参数

Query  查询参数

 token   必填  string(512)

【获取发票下载信息 】download_url中的token字段，保留即可。不要删除，也不用额外再添加一遍

 mchid   必填  string(32)

商户号

 sub_mchid   必填  string(32)

子商户号，服务商模式必须填写

 openid   必填 string(32)

通过查询接口获取的card_openid

 invoice_code   选填  string(12)

发票代码

 invoice_no   必填  string(20)

发票号码

 fapiao_id   必填  string(32)

唯一标识一张发票

file_format  选填  string

发票文件格式，不填时默认为PDF格式

可选取值

  • PDF: PDF格式

  • XML: XML格式

  • OFD: OFD格式

当获取到账单文件资源的下载地址信息后，如：
