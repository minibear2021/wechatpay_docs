商户通过[创建支付分订单](https://pay.weixin.qq.com/doc/v3/partner/4013138534.md)接口获取确认订单的必要参数package后可使用微信支付提供的JS调起微信支付分小程序，引导用户确认订单（公众号端）

## 接口说明

支持商户： 【普通服务商】

接口名称： openBusinessView

提示

- 此接口引用 JSAPI版本1.5.0，引用地址： [https://res.wx.qq.com/open/js/jweixin-1.5.0.js](https://res.wx.qq.com/open/js/jweixin-1.5.0.js)

- 要求用户微信版本>=7.0.5


## 字段说明

### 请求参数

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| businessType | string(16) | true | 固定值，请传入wxpayScoreUse。 |
| queryString | string(2048) | true | 使用URL的query string方式传递参数，格式为key=value&key2=value2，其中value，value2需要进行UrlEncode处理。 |
| queryString.package | string(128) | true | 【跳转支付分小程序订单数据】创建支付分订单成功后返回，用于拉起支付分小程序确认订单页面，由数字大小写字母\_-符号组成，不超过300字符。 |
| err_msg | string(128) | true | 返回信息。 |
| extraData | Object | false | 当err\_code为0时，extraData才返回 |
| extraData.query_id | string(64) | true | 单据查询ID，对应 [查询支付分订单](https://pay.weixin.qq.com/doc/v3/partner/4013138559.md) 接口中入参query\_id。 |
| extraData.appid | string(32) | true | 支付分小程序appid，固定值wxd8f3793ea3b935b8。 |

 /**
  * 版本号比较
  * @param {string
} v1
  * @param {string
} v2
  */
function compareVersion(v1, v2) {
    v1 = v1.split('.')
    v2 = v2.split('.')
    const len = Math.max(v1.length, v2.length)

    while (v1.length < len) {
      v1.push('0')
    }
    while (v2.length < len) {
      v2.push('0')
    }

    for (let i = 0; i < len; i++) {
      const num1 = parseint(v1[i])
      const num2 = parseint(v2[i])

      if (num1 > num2) {
        return 1
      } else if (num1 < num2) {
        return -1
      }
    }

    return 0
 }
