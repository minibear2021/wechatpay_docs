## 1、员工权限管理说明

服务商或子商户（也叫特约商户）商户号的超级管理员可在[服务商平台](https://pay.weixin.qq.com/partner/public/home)或[商户平台](https://pay.weixin.qq.com/)进行员工权限管理（新增员工账号，新增角色权限等）。具体操作步骤请参考下方指引。

注意

原超级管理员已离职/失联，可参考[修改超级管理员指引](https://kf.qq.com/faq/2008047vm2ey200804ruiMFf.html)更换。

## 2、服务商员工权限管理

### 2.1、新增角色

服务商商户号超级管理员登录[服务商平台](https://pay.weixin.qq.com/partner/public/home)，通过路径【账户中心 -> 账户设置 -\> 进入“员工账户管理” 】页面来新增角色或对已有角色进行编辑。新增或修改角色时，必须为该角色配置登录商户平台的权限，否则无法登录平台进行后续操作。

| 步骤一：登录服务商平台，点击“账户中心-账户设置-员工账户管理” | 步骤二：点击“新增角色”选择对应权限 |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/31809801ac5ed88417674663167b454e.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/8592ee8e38ab21be3f2aff694645c0d2.png) |

### 2.2、新增员工账号并为其分配角色

服务商商户号超级管理员登录[服务商平台](https://pay.weixin.qq.com/partner/public/home)，通过路径【账户中心 -> 账户设置 -> 员工账户管理 -\> 进入“员工列表” 】页面点击“新增员工账号”并填写员工信息，填写信息时在“角色权限”字段为其赋予配置了对应权限的角色。

| 步骤一：员工列表，点击“新增员工账号” | 步骤二：填写员工信息，并为其赋予角色权限 |
| :-- | :-- |

## 3、子商户员工权限管理

### 3.1、新增角色

子商户号超级管理员登录[商户平台](https://pay.weixin.qq.com/)，通过路径【账户中心 -\> 进入“员工账户管理” 】页面来新增角色或对已有角色进行编辑。新增或修改角色时，必须为该角色配置登录商户平台的权限，否则无法登录平台进行后续操作。

| 步骤一：登录商户平台，点击“账户中心-员工账户管理” | 步骤二：点击“新增角色”选择对应权限 |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/dce581dc51df5d35191c6a6b2085f0b3.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/66575bc3490f0531244e39f3e241a32b.png) |

### 3.2、新增员工账号并为其分配角色

子商户号超级管理员登录[商户平台](https://pay.weixin.qq.com/)，通过路径【账户中心 -> 员工账户管理 -\> 进入“员工列表” 】页面点击“新增员工账号”并填写员工信息，填写信息时在“角色权限”字段为其赋予配置了对应权限的角色。

| 步骤一：员工列表，点击“新增员工账号” | 步骤二：填写员工信息，并为其赋予角色权限 |
