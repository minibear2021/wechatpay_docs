电商收付通中，针对多次请求出境的场景，商家需要查询可出境金额之后再进行出境请求

其中：订单剩余可出境金额=订单可出境金额-已累计出境金额


## 接口说明

支持商户：【平台商户】

请求方式：【GET】 `/v3/funds-to-oversea/transactions/{transaction_id}/available_abroad_amounts`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/partner/4012082567.md)

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| transaction_id | string(32) | true | **【微信支付订单号】** 微信支付返回的支付订单号 |
### Query 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| sub_mchid | string(32) | true | **【二级商户号】** 查询剩余可出境金额的二级商户号 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| transaction_id | string(32) | true | **【微信支付订单号】** 微信支付返回的支付订单号 |
| available_abroad_amount | integer | true | **【订单剩余可出境金额】** 订单剩余可出境金额，单位: 分 |

		host   = "https://api.mch.weixin.qq.com"

	reqUrl, err := url.Parse(fmt.Sprintf("%s%s", host, path))
	if err != nil {

	if err != nil {
		return nil, err
	}

		}

	} else {

			httpResponse.StatusCode,
