商户通过该接口，打开微信支付分开通小程序，引导用户进行服务开通。 来源小程序的AppID必须与商户号，或者子商户号相关联。

## 接口说明

支持商户： 【普通服务商】

接口名称： wx.navigateToMiniProgram，详见[小程序跳转API](https://developers.weixin.qq.com/miniprogram/dev/api/navigate/wx.navigateToMiniProgram.html)文档

## 接口参数

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| appId | string(32) | true | 车主服务小程序AppID，固定值：wxbcad394b3d99dac9 |
| path | string(64) | true | 小程序页面path，固定值：pages/auth-creditpay/auth-creditpay |
| extraData | object | true | 需要传递给支付分的业务数据。 |
| extraData.mchid | string(32) | true | 商户号。 |
| extraData.sub_mchid | string(32) | true | 子商户号 |
| extraData.openid | string(32) | true | 用户在商户对应AppID下的唯一标识。 |
| extraData.plate_number | string(32) | true | 待开通车牌。 |
| extraData.plate_color | string(32) | true | 待开通车牌的颜色<br>BLUE：蓝色<br>GREEN：绿色<br>YELLOW：黄色<br>BLACK：黑色<br>WHITE：白色<br>LIMEGREEN：黄绿色 |
| extraData.trade_scene | string(32) | false | 开通场景信息，目前只支持PARKING |

示例代码

```
wx.navigateToMiniProgram({
  appId: 'wxbcad394b3d99dac9',
  path: '/pages/auth-creditpay/auth-creditpay',
  extraData: {
    mchid: '1230000100',
    sub_mchid: '1900000109',
    openid: '5K8264ILTKCH16CQ250',
    plate_number: '粤B888888',
    plate_color: 'BLUE',
    trade_scene: 'PARKING'
  },
  success() {
    //dosomething
  },
  fail() {
    //dosomething
  },
  complete() {
    //dosomething
  }
});
```

## 应答参数

用户授权完成之后，会跳转回商户小程序。可通过onShow(OBJECT)所携带的参数判断返回结果，OBJECT返回参数请查看[小程序开发文档onShow参数说明](https://mp.weixin.qq.com/debug/wxadoc/dev/framework/app-service/app.html)

示例代码

```
App({
    onShow(res) {
        if (res.scene === 1038) { // 场景值1038：从被打开的小程序返回
            const { appId, extraData } = res.referrerInfo
            if (appId == 'wxbcad394b3d99dac9') { // appId为wxbcad394b3d99dac9：从车主小程序跳转回来
                // TODO
                // 客户端小程序不确定授权结果，需要发起‘查询车牌服务开通信息’确认授权结果
                return;
            }
        }
    }
})
```
