小程序调起微信支付分停车服务开通页

更新时间：2024.10.31

商户通过该接口，打开微信支付分开通小程序，引导用户进行服务开通。 来源小程序的AppID必须与商户号，或者子商户号相关联。

 

接口说明

支持商户： 【普通服务商】

接口名称： wx.navigateToMiniProgram，详见小程序跳转API文档

接口参数

appId必填 string(32)

车主服务小程序AppID，固定值：wxbcad394b3d99dac9

path必填 string(64)

小程序页面path，固定值：pages/auth-creditpay/auth-creditpay

extraData必填 object

需要传递给支付分的业务数据。

属性

mchid必填 string(32)

商户号。

sub_mchid必填 string(32)

子商户号

openid必填 string(32)

用户在商户对应AppID下的唯一标识。

plate_number必填 string(32)

待开通车牌。

plate_color必填 string(32)

待开通车牌的颜色
BLUE：蓝色
GREEN：绿色
YELLOW：黄色
BLACK：黑色
WHITE：白色
LIMEGREEN：黄绿色

trade_scene 选填 string(32)

开通场景信息，目前只支持PARKING

 

示例代码

1wx.navigateToMiniProgram({
2  appId: 'wxbcad394b3d99dac9',
3  path: '/pages/auth-creditpay/auth-creditpay',
4  extraData: {
5    mchid: '1230000100',
6    sub_mchid: '1900000109',
7    openid: '5K8264ILTKCH16CQ250',
8    plate_number: '粤B888888',
9    plate_color: 'BLUE',
10    trade_scene: 'PARKING'
11  },
12  success() {
13    //dosomething
14  },
15  fail() {
16    //dosomething
17  },
18  complete() {
19    //dosomething
20  }
21});

应答参数

用户授权完成之后，会跳转回商户小程序。可通过onShow(OBJECT)所携带的参数判断返回结果，OBJECT返回参数请查看小程序开发文档onShow参数说明

示例代码

1App({
2    onShow(res) {
3        if (res.scene === 1038) { // 场景值1038：从被打开的小程序返回
4            const { appId, extraData } = res.referrerInfo
5            if (appId == 'wxbcad394b3d99dac9') { // appId为wxbcad394b3d99dac9：从车主小程序跳转回来
6                // TODO
7                // 客户端小程序不确定授权结果，需要发起‘查询车牌服务开通信息’确认授权结果
8                return;
9            }
10        }
11    }
12})
