用于在顾客下单后为顾客分配服务人员使用。

注意：

- 调用服务人员分配接口需在完成支付之前，若分配服务人员晚于支付完成，则将无法在支付凭证上出现服务人员名片入口。





## 接口说明

支持商户：【普通服务商】

请求方式：【POST】 `/v3/smartguide/guides/{guide_id}/assign`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/partner/4012082567.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/partner/4012365870.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `application/json` |
### Path 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| guide_id | string(64) | true | **【服务人员ID】** 导购在导购系统中的唯一标识 |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| sub_mchid | string(64) | false | **【子商户ID】** 导购所属商户的商户ID |
| out_trade_no | string(32) | true | **【商户订单号】** 商户系统内部订单号，要求32个字符内，只能是数字、大小写字母\_-&#124;\* ，且在同一个商户号下唯一 |
