服务商为特约商户开通或关闭点金计划。




## 接口说明

支持商户：【普通服务商】　【渠道商】

请求方式：【POST】 `/v3/goldplan/merchants/changegoldplanstatus`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/partner/4012082567.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/partner/4012365870.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `application/json` |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| sub_mchid | string(32) | true | **【特约商户号】** 开通或关闭点金计划的特约商户商户号，由微信支付生成并下发。 |
| operation_type | string | true | **【操作类型】** 开通或关闭点金计划的动作，枚举值：<br>OPEN：表示开通点金计划<br>CLOSE：表示关闭点金计划<br>可选取值：<br><ul><li>`OPEN`: 开通</li></ul><br><ul><li>`CLOSE`: 关闭</li></ul> |
| operation_pay_scene | string | false | **【支付场景】** 支付场景，指定开通/关闭点金计划的支付场景。如果未指定，则默认值是JSAPI<br>可选取值：<br><ul><li>`JSAPI_AND_MINIPROGRAM`: jsapi支付和小程序支付两个场景</li></ul><br><ul><li>`JSAPI`: jsapi支付场景</li></ul><br><ul><li>`MINIPROGRAM`: 小程序支付场景</li></ul> |
