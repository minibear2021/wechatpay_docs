## 1、申请开通合单支付产品权限

开通合单支付产品权限有两步：

- 服务商(合单发起方)申请合单支付权限，[详见1.1](https://pay.weixin.qq.com/doc/v3/partner/4013461849.md#1.1%E3%80%81%E7%94%B3%E8%AF%B7%E5%90%88%E5%8D%95%E6%94%AF%E4%BB%98%E6%9D%83%E9%99%90)。

- 服务商(合单发起方)需要开通相应场景的单笔支付产品权限、子商户(所有子单参与方)需要授权服务商相应场景的支付产品权限，[详见1.2](https://pay.weixin.qq.com/doc/v3/partner/4013461849.md#1.2%E3%80%81%E5%BC%80%E9%80%9A%E7%9B%B8%E5%BA%94%E5%9C%BA%E6%99%AF%E7%9A%84%E6%94%AF%E4%BB%98%E4%BA%A7%E5%93%81%E6%9D%83%E9%99%90)。


### 1.1、申请合单支付权限

服务商需要联系对接的运营人员申请对应合单支付权限（为服务商申请发起合单权限，为子商户申请接收合单权限）。

### 1.2、开通相应场景的支付产品权限

服务商(合单发起方)需要开通相应场景的单笔支付产品权限，子商户(所有子单参与方)需要授权服务商相应场景的单笔支付产品权限，具体指引请参考下方表格：

| 产品 | 开通相应场景的单笔支付产品权限 |
| :-- | :-- |
| [APP合单支付](https://pay.weixin.qq.com/doc/v3/partner/4012079331.md) | 参考[APP支付权限申请](https://pay.weixin.qq.com/doc/v3/partner/4013080228.md) |
| [H5合单支付](https://pay.weixin.qq.com/doc/v3/partner/4013462080.md) | 参考[H5支付权限申请](https://pay.weixin.qq.com/doc/v3/partner/4012074917.md) |
| [JSAPI合单支付](https://pay.weixin.qq.com/doc/v3/partner/4012079332.md) | 参考[JSAPI支付权限申请](https://pay.weixin.qq.com/doc/v3/partner/4012069853.md) |
| [Native合单支付](https://pay.weixin.qq.com/doc/v3/partner/4012079333.md) | 参考[Native支付权限申请](https://pay.weixin.qq.com/doc/v3/partner/4012076268.md) |
