>更新时间：2026.06.29

对商户提供查询停车场信息接口，返回停车场基础信息、启用状态及未启用/下线原因。适用对象与普通服务商一致，支持普通商户、普通服务商、平台商户（电商模式），与既有停车提醒 API 相同。




## 接口说明

支持商户：【普通服务商】　【平台商户】

请求方式：【GET】 `/v3/parking/reminders/parking-lot`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/merchant/4012365342.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
### Query 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| out_parking_lot_id | string(64) | false | 【商户停车场ID】 商户侧自定义的停车场唯一标识，在「提交停车场进件申请」时传入。仅支持字母、数字、下划线、连字符，长度1~64字符，同一商户号下不可重复。与微信停车场ID二选一，至少传一个；若与微信停车场ID同时传入，以微信停车场ID为准。 |
| wx_parking_lot_id | string(64) | false | 【微信停车场ID】 停车场进件审核通过且运营入驻完成后，由微信支付分配。可通过「查询停车场进件申请单」或「查询停车场信息」接口获取；入场/离场/支付等接口须与入场时保持一致。与商户停车场ID二选一，至少传一个；若与商户停车场ID同时传入，优先使用本字段。 |
