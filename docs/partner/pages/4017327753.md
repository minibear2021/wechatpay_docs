查询图片生成任务执行结果




## 接口说明

支持商户：【普通服务商】

请求方式：【GET】 `/v3/marketing/partner/product-coupon/image-generation-tasks/{task_id}`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

接口限频：20/秒（商户号维度）

## 请求参数

| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/merchant/4012365342.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
### Path 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| task_id | string | true | 【图片生成的任务ID】 服务商创建图片生成任务的请求流水号，服务商侧需保持唯一性，可使用 数字、大小写字母、下划线 `_`、短横线 `-` 组成，长度在6-40个字符之间 |
### Query 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| brand_id | string | true | 【品牌ID】 微信支付为品牌方分配的唯一标识，该品牌应与服务商存在授权关系 |
| image_generation_type | string | true | 【图片生成类型】 与提交图片生成任务时所传的类型一致<br>可选取值<br><ul><li>`COMBINE_IMAGE`:  适合全场类优惠头图，将品牌元素与商品券优惠等信息拼接生成头图</li><li>`CUT_OUT`:  适合单品类优惠头图，用于去除图片背景，保留商品主体</li></ul> |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| brand_id | string | true | 【品牌ID】 微信支付为品牌方分配的唯一标识，该品牌应与服务商存在授权关系 |
| task_id | string | true | 【图片生成的任务ID】 服务商创建图片生成任务的请求流水号，服务商侧需保持唯一性，可使用 数字、大小写字母、下划线 `_`、短横线 `-` 组成，长度在6-40个字符之间 |
| image_generation_type | string | true | 【图片生成类型】 图片生成类型<br>可选取值<br><ul><li>`COMBINE_IMAGE`:  适合全场类优惠头图，将品牌元素与商品券优惠等信息拼接生成头图</li><li>`CUT_OUT`:  适合单品类优惠头图，用于去除图片背景，保留商品主体</li></ul> |
| task_state | string | true | 【任务状态】 任务状态<br>可选取值<br><ul><li>`PROCESSING`:  任务执行中</li><li>`FAIL`:  任务失败</li><li>`SUCCESS`:  任务成功</li></ul> |
| combine_image_result | object | false | 【拼图结果】 当 `image_generation_type` 为 `COMBINE_IMAGE` ，并且任务状态为 `SUCCESS` 时，该数据结构值有效 |
| combine_image_result.image_url | string | true | 【生成的图片URL】 当任务状态为 `SUCCESS` 时，返回该值，其他情况下不会返回 |
| cut_out_result | object | false | 【生成商品图结果】 当 `image_generation_type` 为 `CUT_OUT` ，并且任务状态为 `SUCCESS` 时，该数据结构值有效 |
| cut_out_result.image_url | string | true | 【生成的图片URL】 当任务状态为 `SUCCESS` 时，返回该值，其他情况下不会返回 |
