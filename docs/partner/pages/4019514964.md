## 产品概述

为提升通行效率，微信支付在停车场景推出停车便捷缴费服务。用户通过微信车主服务绑定车牌并启用本服务后，驾车进入已启用本服务的停车场，微信支付将推送停车进场消息给用户。用户可通过消息，查询实时停车状态，在线支付停车费。用户一次开通，所有接入本能力的停车场均可使用。

## 名词解释

| 名词 | 解释说明 |
| --- | --- |
| 停车便捷缴费 | 基于用户微信账户与车牌的绑定关系，用户进入接入车场后通过进场消息推送触达用户，提供缴费入口，实现“消息内提前缴费”的停车支付服务。 |
| 停车进场消息 | 车辆进入接入车场后微信车主服务在微信支付公众号推送的通知，消息承载实时费用与缴费入口，是核心触达载体。 |
| 停车跳价提醒 | 部分接入车场支持的高阶功能，当用户在接入停车场启用了这一功能时，微信车主服务会在停车费达到下一阶梯价格时，向用户推送费用变动提醒。默认关闭，需用户主动在当次停车详情页启用。 |
| 已启用停车场 | 即支持“停车便捷缴费”能力的停车场，车辆进入该类车场时，微信支付会推送进场提醒消息；产生费用后，用户可在该消息内直接完成缴费。 |
| 微信车主服务 | 微信车主服务是微信支付基于用户微信账户与车辆的绑定关系，通过车辆识别进行扣费的支付服务平台，目前提供停车、高速、加油场景的支付服务，停车便捷缴费是微信车主服务平台下的一种服务能力。 |

## 产品优势

1、实时停车状态：用户在进场后立即被推送进场消息，消息内可查看实时费用情况；

2、在线缴停车费：用户可在进场消息内提前缴停车费，全程免扫码；

## 产品流程

### 用户开通流程

目前采取邀约制开通，目标用户在某一已启用服务车场停车后，会收到一条微信车主服务推送的邀约消息，点击即可完成车辆授权和服务开通。开通后，用户进入任意接入车场，都可通过进场消息提前缴停车费。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/80927d1c40dcc470c9a905c1aa23b876.png)

### 用户使用流程

开通用户车辆驶入接入车场时，车主服务会向用户推送进场消息，消息内可查询实时停车状态、提前缴停车费，离场时自动抬杆放行。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/84b875b581d458dcda258d8855a52fc6.png)

### 用户管理流程

用户可在微信车主服务小程序进行车牌服务管理。
