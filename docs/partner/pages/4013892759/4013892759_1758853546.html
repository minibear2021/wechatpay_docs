商户申请单号查询申请单状态

更新时间：2025.09.26

电商平台发起申请后, 电商平台调用此接口查询审批和出款进度

接口说明

支持商户：【平台商户】

请求方式：【GET】/v3/ecommerce/account/apply-cancel-withdraw/out-request-no/{out_request_no}

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点 ，指引点击查看

请求参数

Header  HTTP头参数

 Authorization 必填　string

请参考签名认证生成认证信息

 Accept 必填　string

请设置为application/json

path  路径参数

 out_request_no 必填   string(32)

【商户注销申请单号】 由商户自定义生成的、在提交接口传入的单号

【注销提现申请单状态】 发起注销提现申请后，整个流程可能流转的状态

  • ACCEPTED:  已受理

  • REVIEWING:  审核中

  • WAITING_MERCHANT_CONFIRM:  等待商户确认

  • REVOKED:  已撤销

  • SYSTEM_PROCESSING:  系统处理中

  • FINISH:  注销完成
