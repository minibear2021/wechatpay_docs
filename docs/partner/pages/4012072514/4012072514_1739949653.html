小程序左上角返回键管理

更新时间：2025.02.19

服务商可根据服务商维度、特约商户维度，订单号维度配置该接口

接口说明

支持商户：【普通服务商】【渠道商】

请求参数

payCompletedPageOptions.showNavBackButton 选填 boolean

【是否显示支付回调页左上角返回按钮】 是否显示左上角返回按钮，默认是true

显示和隐藏左上角返回按钮

1wx.requestPayment({
2    payCompletedPageOptions: {
3        showNavBackButton: false // 是否显示左上角返回按钮，默认是 true 
4    }
5})

左上角返回按钮跳转页面配置

1wx.requestPayment({
2    payCompletedPageOptions: {
3        showNavBackButton: true
4    },
5    success: res => {
6        res.payCompletedPage.onUnload(() => {
7                wx.navigateTo({
8                        url: '/page/index/index' // 点击左上角返回按钮，会触发支付回调页onUnload 事件，在 onUnload 事件触发时可以跳转到别的页面
9                })
10        })
11    }
12})
