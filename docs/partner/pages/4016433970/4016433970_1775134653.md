## 1、商家名片配置权限管理

商家名片需要完成名片的内容配置，并且将品牌的交易的小程序或商户号和品牌完成关联，才能实现交易连接名片。由于名片内容直接对用户展示，且会影响品牌支付链路的头像、名称、凭证跳转，属于敏感权限。品牌可针对商家名片设置相应管理岗位，为管理商家名片的运营人员添加该岗位，方可配置商家名片。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/6d96d250e5d3bea005c176a8007a1eae.jpeg)

## 2、商家名片初次配置

- 创建商家名片，需要在「品牌经营平台-名片管理」中完成名片基础信息的配置，以及名片服务列表的配置。

- 配置完成并通过审核后，商家可以操作发布名片。


![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/9817c2705cfdbe920191dc28478fb846.jpeg)

### 2.1、名片基础信息配置

（1）品牌信息：品牌ID的名称和头像，如需修改需要前往「品牌信息」

（2）商家客服：在线客服和客服电话二选一（目前尚未在名片上向用户展示客服按钮，后续会迭代上线）

- 在线客服：提供四种在线客服类型。企业微信客服需提供客服账号链接、小程序官方客服组件需提供appid、商家自建客服网页需提供网页链接、商家自建客服内嵌在小程序需提供appid和小程序路径

- 客服电话：输入官方客服电话


（3）场景跳转：连接品牌在微信内的交易场/内容场，选择希望展示的小程序和视频号，同时支持配置说明文案和图片，丰富内容展示

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/e0a76caeaa8268c8c17742a7e7df0d32.jpeg)

### 2.2、名片服务配置

（1）添加服务：配置小程序跳转和网页跳转均可，跳转到品牌自己的页面。小程序跳转需要配置小程序appid和小程序路径

（2）添加服务分类：给同类型的服务归类，便于用户理解

（3）列表排序：可以拖动某个服务或者是某个服务分类进行列表排序

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/7203e332da1c20f09bed5240fdb05ef9.jpeg)

## 3、交易连接名片

交易连接名片配置，用户在商家支付成功后，支付链路将展示品牌的头像和名称，由微信支付公众号下发的支付凭证，将支持直接跳转商家名片。不同的支付产品，配置的规则不同，目前已支持配置小程序支付、App支付、付款码支付和支付分。

|     |     |
| --- | --- |
| 支付产品 | 关联帐号 |
| 小程序支付 | 用户下单完成付款的小程序 |
| App支付 | 用户下单完成付款的App，在微信支付下单接口中传入的appid |
| 付款码支付 | 商家收款的商户号 |
| 支付分 | 服务ID |

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/1679d7b3639b517a969aa381bbf3782d.jpeg)![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/63f7230b2e0e7369a95506c8f8610489.jpeg)

### 3.1、小程序支付

页面将会展示品牌已经绑定的小程序AppID，请勾选用户实际发起交易的小程序，点击确认添加后，该小程序发起的交易将连接商家名片

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/34bc80be4bbbeb97b0f9c78fa395b24f.jpeg)

### 3.2、APP支付

页面将会展示品牌已经绑定的App，请勾选用户实际发起交易的App，点击确认添加后，该App发起的交易将连接商家名片

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/4264ef074c65b401bad7a4c5964b21e7.jpeg)

### 3.3、付款码支付

页面将会展示品牌已经绑定的商户号，请勾选在付款码场景下商户实际收款的商户号，点击确认添加后，该商户号收款的交易将连接商家名片。提供两种上传方式，商户号较少可以直接页面勾选；商户号较多，可以下载品牌的商户号列表，内部确认后，将商户号列表重新上传文件确认添加。请注意上传文件时，文件中的商户号需要是品牌已经绑定的商户号。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/72648c5c9768b31eba7ca20ce5d7858b.jpeg)

### 3.4、支付分

请输入支付分入驻时平台分配的服务ID（部分服务商模式暂不支持添加），服务ID下绑定的商户号至少有一个需要在品牌中的商户号列表中，才可以添加成功

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/698e8ace2bea79f7dc0415fea0aac0b5.jpeg)
