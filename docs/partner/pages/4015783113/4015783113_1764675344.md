根据品牌门店ID，删除品牌门店。


## 接口说明

支持商户：【普通服务商】

请求方式：【DELETE】 `/v3/brand/partner/store/brandstores/{store_id}`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

接口限频：900/分钟（商户号维度）

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/merchant/4012365342.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
| Content-Type | string | true | 请设置为 `application/json` |
### Path 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| store_id | string | true | 【品牌门店ID】 创建品牌门店后，系统为该门店分配的唯一ID。 |
### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| brand_id | string | true | 【品牌ID】 商家进驻微信支付品牌商家后获得的品牌ID。 |
## 应答参数

无应答包体

func DeleteBrandStore(config *wxpay_utility.MchConfig, request *DeleteBrandStoreRequest) (err error) {
	const (
		host   = "https://api.mch.weixin.qq.com"
		method = "DELETE"
		path   = "/v3/brand/partner/store/brandstores/{store_id}"
	)

	reqUrl, err := url.Parse(fmt.Sprintf("%s%s", host, path))
	if err != nil {
		return err
	}
	reqUrl.Path = strings.Replace(reqUrl.Path, "{store_id}", url.PathEscape(*request.StoreId), -1)

	if err != nil {

	httpRequest, err := http.NewRequest(method, reqUrl.String(), bytes.NewReader(reqBody))
	if err != nil {
		return err

	httpRequest.Header.Set("Accept", "application/json")

		return err

	if err != nil {
		return err
	}

		return err

			config.WechatPayPublicKey(),

			return err
		}
		return nil
	} else {
		return wxpay_utility.NewApiException(
			httpResponse.StatusCode,
			httpResponse.Header,
			respBody,
		)
	}
}
