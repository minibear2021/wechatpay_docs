## 1、平台下载账单概览

服务商与子商户（也叫特约商户）的超级管理员可自主在[服务商平台](https://pay.weixin.qq.com/partner/public/home)或[商户平台](https://pay.weixin.qq.com/)下载账单。此外，超级管理员还可创建员工账号并分配具备下载对账单权限的角色（详见：[平台员工权限管理](https://pay.weixin.qq.com/doc/v3/partner/4013080349.md)），授权员工进行账单下载操作。

## 2、操作步骤说明

### 类型一：平台下载交易账单

平台下载交易账单有两种方式：

1、服务商登录[服务商平台](https://pay.weixin.qq.com/partner/public/home)，通过路径【交易中心 -> 账单管理 -\> 进入“交易账单”】页面进行交易账单的下载。（只能下载服务商旗下所有子商户的交易账单）

2、子商户登录[商户平台](https://pay.weixin.qq.com/)，通过路径【交易中心 -\> 进入“交易账单”】页面进行交易账单的下载。（可下载对应服务商商户号下自己的交易账单）

| 服务商下载旗下所有子商户的交易账单 | 子商户下载对应服务商商户号下自己的交易账单 |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/07aee18fe82332752db1fec5a8fe2d6a.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/49a67fca6f1c56bc8dc9d82e11d1b01f.png) |
| 账单筛选条件 |
| 对账单归属 | 选择“服务商商户号”。 |
| 对账单类型 | 选择需下载的订单类型，其中：<br>- 所有交易账单（不含可用余额退款）：所有支付成功的订单以及商户退款受理成功的退款。<br>  <br>- 支付账单：所有支付成功的订单。<br>  <br>- 退款账单（不含可用余额退款）：所有商户退款受理成功的退款单。<br>  <br>- 可用余额退款对账单：可用余额的退款单。 |
| 对账单时间 | 选择需要下载的时间段。时间跨度最多支持31天。 |
| office使用版本 | 选择非Office2003或Office2003，选择后点击打包下载即可。 |

注意：

3个月之前账单不支持实时下载，可预约后等待下载结果。

### 类型二：平台下载资金账单

平台下载资金账单有两种方式：

1、服务商登录[服务商平台](https://pay.weixin.qq.com/partner/public/home)，通过路径【交易中心 -> 账单管理 -\> 进入“资金账单”】页面进行资金账单的下载。（只能下载服务商自己的资金账单）

2、子商户登录[商户平台](https://pay.weixin.qq.com/)，通过路径【交易中心 -\> 进入“资金账单”】页面进行资金账单的下载。（只能下载子商户自己的资金账单）

| 服务商下载自己的资金账单 | 子商户下载自己的资金账单 |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/88804cf506cec86949aa270bf38a8aa3.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/cd5eebd88e030f3547ede39f74609179.png) |
| 账单筛选条件 |
| 账户类型 | 选择需下载的资金流水所属账户：<br>- 基本账户：主要用于交易收款，收款资金可进行退款、提现、分账等。<br>  <br>- 运营账户：若需要使用代金券、商家转账等运营业务，则可开通运营账户进行运营业务的出资，便于预算管理、资金对账。 |
| 日期 | 选择需要下载的时间段。时间跨度最多支持31天。 |
| office使用版本 | 选择Office2007以上版本或Office2007以下版本。 |

注意：

平台下载资金账单分两种：

- 业务明细账单：详细展示每一日所有业务资金流水的账单

- 业务汇总账单：按日汇总的账单，展示每日总出入金和余额


### 类型三：查询实时资金流水

#### 1、平台查询实时资金流水

平台查询实时资金流水有两种方式：

1、服务商登录[服务商平台](https://pay.weixin.qq.com/partner/public/home)，通过路径【交易中心 -> 资金管理 -\> 进入“资金流水”】页面进行资金流水的实时查询。

2、子商户登录[商户平台](https://pay.weixin.qq.com/)，通过路径【交易中心  -\> 进入“资金流水”】页面进行资金流水的实时查询。

| 服务商查询自己实时资金流水 | 子商户查询自己实时资金流水 |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/bc281c4f978ef16937417722631456e4.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/e1a8da5e8591a88c776af9c08f33e96e.png) |
| 资金流水筛选条件 |
| 账户类型 | 选择需查询的资金流水所属账户：<br>- 基本账户：主要用于交易收款，收款资金可进行退款、提现、分账等。<br>  <br>- 运营账户：若需要使用代金券、商家转账等运营业务，则可开通运营账户进行运营业务的出资，便于预算管理、资金对账。 |
| 日期 | 选择需要查询的时间段。时间跨度最多支持31天。 |

注意：

仅支持近一个月内的资金流水记录的实时查询。

实时资金流水仅支持页面查询，不支持下载。如需本地查看资金流水信息请通过下载资金账单获取。

#### 2、商家助手小程序查询实时资金流水

服务商与子商户的超级管理员可登录【微信支付商家助手小程序】查询自己的资金流水。此外，超级管理员还可创建员工账号，授权员工查询实时资金账单的权限。

1、登录商家助手

微信客户端中搜索【微信支付商家助手小程序】，进入后选择对应商户号登录。

2、查看资金流水

超级管理员或已配置权限的员工登录小程序后，在小程序首页选择【资金管理】，进入后选择需查看流水的账户，在账户详情页点击下方的【资金明细】即可查看。

3、配置员工权限

超级管理员还可创建员工账号，授权员工查询实时资金账单的权限，详细步骤如下：

步骤一：在小程序首页选择【员工管理 \> 添加员工】，进入后可转发给员工或让其扫码填写对应信息。

步骤二：员工提交申请后，再次进入【员工管理】，点击【待审核员工】，并为其赋予【资金管理】权限。
