## 1、查询商户号绑定APPID账号

### 1.1、查询服务商自身绑定的APPID账号（即combine\_appid）

服务商可登录[服务商平台](https://pay.weixin.qq.com/index.php/partner/public/home)，通过路径【产品中心 -\> APPID账号管理 -> 我关联的APPID账号】来查看所有绑定的APPID账号。

| 查询服务商自身绑定的APPID |
| :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/efd1aa570c8665e47bf1abce12456a9a.png) |

### 1.2、查询服务商为子商户（也叫特约商户）绑定的APPID账号（即sub\_appid）

查询服务商为子商户绑定的APPID账号有两种方式：

#### 1.2.1、服务商查询

服务商查询自身为子商户绑定的APPID账号，服务商可登录[服务商平台](https://pay.weixin.qq.com/index.php/partner/public/home)，通过路径【合作伙伴功能 -> 开发参数配置 -> 找到对应子商户号点击“开发配置” -\> 特约商户APPID配置】来查看所有服务商为子商户绑定的APPID。

| 步骤一：登录服务商平台，点击“合作伙伴功能-开发参数配置-开发配置” | 步骤二：查看所有服务商为子商户绑定的APPID |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/27907068d05fd92d4e7ab53e1f77c4ca.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/ae360bfd5c347ca137dc6281910e039d.png) |

#### 1.2.2、子商户查询

子商户查询服务商为其绑定的APPID账号，子商户可登录[商户平台](https://pay.wechatpay.cn/)，通过路径【产品中心 -\> APPID账号配置 -> 服务商为我关联的APPID账号 】来查看所有服务商为子商户绑定的APPID。

| 查看所有服务商为子商户绑定的APPID |

## 2、商户号绑定APPID账号操作指南

在开发合单支付前，需进行商户号与APPID账号的绑定操作，详细的操作指引请参见下方表格：

【合单服务商商户号】 `combine_mchid` 要与【合单服务商APPID】 `combine_appid` 完成绑定；

【子商户号】 `sub_orders.sub_mchid` 要与【子商户APPID】 `sub_orders.sub_appid` 完成绑定。

| 产品 | 可使用的APPID类型 | APPID申请平台 | 绑定流程 |
| :-- | :-- | :-- | :-- |
| APP合单支付 | 移动应用 | [开放平台](http://open.weixin.qq.com/)申请移动应用 | 服务商为自身绑定APPID请参考[服务商商户号与AppID账号关联管理](https://kf.qq.com/faq/200520fueQrI200520aQf6Rr.html)。<br>服务商为子商户绑定APPID请参考[服务商为子商户配置APPID(即sub\_appid)](https://kf.qq.com/faq/190715yaYnYv1907153mmIbA.html)。 |
| JSAPI合单支付 | 公众号 | [公众平台](http://mp.weixin.qq.com/)申请公众号 |
| 小程序合单支付 | 小程序 | [公众平台](http://mp.weixin.qq.com/)申请小程序 |
| H5合单支付 | 移动应用/公众号/小程序 | [开放平台](http://open.weixin.qq.com/)申请移动应用或[公众平台](http://mp.weixin.qq.com/)申请公众号/小程序 |
