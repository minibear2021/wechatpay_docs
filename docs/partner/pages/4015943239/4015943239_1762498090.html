常见问题

更新时间：2025.11.07

Q：平台收付通注销二级商户号接口报错：{"code":"INVALID_REQUEST","message":"该商户号账户有余额，请先提现后再申请注销"}

A：需要先使用二级商户预约提现接口将二级商户号中的余额全部提现后再申请注销。

Q：调用提交注销申请单接口报错： {"code":"PARAM_ERROR"，"detail":{"location":null;"value":1};"message":"输入源“[/body/application_info/0]”映射到字段“注销申请材料”多重性规则校验失败，此字段个数为 1 个，低于最小值 2"}。

A：application_info为数组类型，需要上传两种注销申请材料，请参考注销申请材料。

Q：调用提交注销申请单接口报错：{"code": "PARAM ERROR","message":“无法将 JSON 输入源 “/body/application_info”映射到多重字段“注销申请材料”中，此字段需要 JSON 数组对象”}。

A：application_info 参数需为 JSON 数组对象。请检查请求体中该字段是否以数组形式传递，例如：

1"application_info" : [
2    {
3      "application_type" : "SP_CANCEL_ACCOUNT_APPLICATION",
4      "application_media_id" : "abc123456"
5    }
6 ]

若仅包含单个对象，请用方括号包裹。确保数组结构符合接口规范，避免单个对象直接赋值。
