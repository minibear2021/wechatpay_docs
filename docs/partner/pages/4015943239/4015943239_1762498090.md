## Q：平台收付通注销二级商户号接口报错：{"code":"INVALID\_REQUEST","message":"该商户号账户有余额，请先提现后再申请注销"}

A：需要先使用[二级商户预约提现](https://pay.weixin.qq.com/doc/v3/partner/4012476652.md)接口将二级商户号中的余额全部提现后再申请注销。

## Q：调用提交注销申请单接口报错： {"code":"PARAM\_ERROR"，"detail":{"location":null;"value":1};"message":"输入源“\[/body/application\_info/0\]”映射到字段“注销申请材料”多重性规则校验失败，此字段个数为 1 个，低于最小值 2"}。

A：application\_info为数组类型，需要上传两种注销申请材料，请参考[注销申请材料](https://docs.qq.com/doc/p/cc53548f88238e7d58337f733b4e2997c4fb3253)。

## Q：调用提交注销申请单接口报错：{"code": "PARAM ERROR","message":“无法将 JSON 输入源 “/body/application\_info”映射到多重字段“注销申请材料”中，此字段需要 JSON 数组对象”}。

A： `application_info` 参数需为 JSON 数组对象。请检查请求体中该字段是否以数组形式传递，例如：

```
"application_info" : [
    {
      "application_type" : "SP_CANCEL_ACCOUNT_APPLICATION",
      "application_media_id" : "abc123456"
    }
 ]
```

若仅包含单个对象，请用方括号包裹。确保数组结构符合接口规范，避免单个对象直接赋值。
