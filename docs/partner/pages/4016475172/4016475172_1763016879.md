服务商可调用本接口撤销处于非发布状态的配置申请。




## 接口说明

支持商户：【普通服务商】

请求方式：【POST】 `/v3/brand/card/card-configs/cancel-applyment`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

接口限频：1/秒（商户号维度）

| brand_id | string(20) | true | 【品牌ID】 商家进驻微信支付品牌商家后获得的品牌ID，作为品牌唯一标识符。 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| business_code | string(128) | true | 【业务申请编号】 1、只能由数字、字母或下划线组成；<br>2、服务商自定义的商户唯一编号，每个编号对应一个申请单。 |
| applyment_id | string(25) | true | 【微信支付申请单号】 申请单的唯一标识，由微信支付生成。 |
| brand_id | string(20) | true | 【品牌ID】 商家进驻微信支付品牌商家后获得的品牌ID，作为品牌唯一标识符。 |
| applyment_state | string | true | 【申请单状态】 商家名片配置的申请状态，如果撤销成功，该字段只会返回 `CANCELED`。<br>可选取值<br><ul><li>`STATE_UNKNOWN`:  「状态未知，非终态」发布配置流程的中间状态，请稍后重试</li><li>`DRAFTING`:  「配置中，非终态」调用提交申请接口后，返回了预览二维码，尚未调用发布接口，即流转到该状态</li><li>`AUDITING`:  「审核中，非终态」调用发布接口后，平台将进行审核，即流转到该状态</li><li>`AUDIT_REJECTED`:  「审核驳回，非终态」审核阶段被驳回，会附带具体的驳回原因，请根据驳回原因修改调整，如仍需添加请使用原业务申请编号重新调用 [提交商家名片配置申请](https://pay.weixin.qq.com/doc/v3/partner/4016468440.md) 接口</li><li>`PENDING_PUBLISH`:  「待发布，非终态」审核通过后，未到达发布时间，即流转到该状态</li><li>`PUBLISHED`:  「已发布，终态」商家名片配置已顺利更新到线上，即流转到该状态</li><li>`CANCELED`:  「已撤销，终态」商家名片配置请求被撤销，如仍需配置请更换业务申请编号重新调用 [提交商家名片配置申请](https://pay.weixin.qq.com/doc/v3/partner/4016468440.md) 接口</li></ul> |
