收付通平台可通过此接口查询本商户号指定日期当天24点的账户余额。

注意：

- 可查询90天内的日终余额。

- 当日日终余额在次日生成，建议商户在上午 10 点以后查询。





## 接口说明

支持商户：【平台商户】

请求方式：【GET】 `/v3/merchant/fund/dayendbalance/{account_type}`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/partner/4012082567.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/partner/4012365870.md#%E8%AF%B7%E6%B1%82%E6%8E%A5%E5%8F%A3%E5%90%8E%E7%AB%AF%E7%AD%BE%E5%90%8D) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
### Path 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| account_type | string | true | **【账户类型】** 账户类型<br>可选取值：<br><ul><li>`BASIC`: 基本户</li></ul><br><ul><li>`OPERATION`: 运营账户</li></ul><br><ul><li>`FEES`: 手续费账户</li></ul> |
### Query 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| date | string(10) | false | **【日期】** 日期，格式YYYY-MM-DD |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| available_amount | integer | true | **【可用余额（单位：分）】** 可用余额（单位：分），此余额可做提现操作 |
| pending_amount | integer | false | **【不可用余额（单位：分）】** 不可用余额（单位：分） |
