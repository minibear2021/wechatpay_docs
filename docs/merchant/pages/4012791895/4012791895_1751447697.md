小程序支付和JSAPI支付，共用一个权限，小程序支付场景下，只需要按照本流程申请JSAPI支付权限。

## 1、已有商户号申请JSAPI支付产品权限

若商户已有商户号(未开通JSAPI支付权限)，需新增JSAPI支付权限，可通过【[商户平台](https://pay.wechatpay.cn/index.php/core/home/login) -> 产品中心 -> JSAPI支付 -> 申请开通】，待审核(7个工作日内)通过后将开通JSAPI支付权限。

| 步骤一：登录商户平台，点击“产品中心” | 步骤二：点击“JSAPI支付” |
| :-- | :-- |

## 2、申请新商户号和JSAPI支付产品权限

若商户没有商户号，请参考[微信支付商户入驻详细流程](https://kf.qq.com/faq/180910eUfeyu180910BFZNJN.html)指引申请商户号，按照下面流程指引入驻成功后，商户号将默认开通JSAPI支付产品权限。

当在“填写经营与行业信息”这一步骤时，经营场景选择小程序，并填写小程序APPID和小程序页面截图。

| 小程序场景申请JSAPI支付产品权限 | 字段填写说明 |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/06173239d841154896486ac94f42030d.png) | 商户简称 | 1、 不支持单纯以人名来命名，若为个体户经营，可用“个体户+经营者名称”或“经营者名称+业务”命名，如“个体户张三”或“张三餐饮店”；<br>2、不支持无实际意义的文案，如“XX特约商户”、“800”、“XX客服电话XXX”；<br>3、填写的商户简称会在交易场景中显示，参考：[入驻填写的商户简称展示方式](https://kf.qq.com/faq/161226RbUVvU161226JjY3M7.html) |
| 客户电话 | 1、请填写真实有效的客服电话，将在交易记录中向买家展示，提供咨询服务。<br>2、请确保电话畅通，以便入驻后微信侧回拨确认。 |
| 所属行业 | 所属行业请参考：[入驻结算规则、行业属性及特殊资质](https://kf.qq.com/faq/220228IJb2UV220228uEjU3Q.html)。 |
| 小程序APPID | 请填写已认证的小程序APPID。要求申请主体一致。<br>1、商户如未申请小程序的appid，可参考指引注册申请：[注册公众平台账号](https://kf.qq.com/product/weixinmp.html#hid=87)。<br>2、商户如已申请小程序的appid，可登录[公众平台](https://mp.weixin.qq.com/)，参考指引[查看小程序APPID](https://pay.weixin.qq.com/doc/v3/merchant/4013287504.md#2.1.2%E3%80%81%E5%B0%8F%E7%A8%8B%E5%BA%8F)。<br>注意：入驻时填写的appid，系统会自动发起商户号与该APPID的授权绑定，商户参考指引[确认授权绑定商户号](https://pay.weixin.qq.com/doc/v3/merchant/4013287504.md#2.3.2%E3%80%81%E5%B0%8F%E7%A8%8B%E5%BA%8F)。 |
| 小程序页面截图<br>（选填） | 请提供展示商品/服务的页面截图/设计稿（最多5张），若小程序未建设完善或未上线请务必提供，图片需在2M内。 |
