## 一、上线前（联调阶段）常见问题

| 检查项 | 详细描述与解决方案 | 参考文档 |
| :-- | :-- | :-- |
| 商户号与appid绑定校验 | 确认service\_id下是否已正确绑定发起调用的 mchid（商户号）和 appid。两者必须都在该服务 ID 的授权范围内。 | [商户号与appid绑定配置指引](https://pay.weixin.qq.com/doc/v3/merchant/4012587233.md) |
| 参数一致性 | 检查接口传入的 mchid、appid 是否与后台配置完全一致（多 AppID 场景易传错参数）。 | [查询商户号绑定的appid](https://pay.weixin.qq.com/doc/v3/merchant/4016328613.md) |
| 模式与权限匹配 | 1. 需确认模式： `need_user_confirm` 必须传 true。<br>2. 免确认模式：属于高级权限，需特殊申请。若未申请时传参 `need_user_confirm = false` 会报权限不足。 | [支付分创建订单指引](https://pay.weixin.qq.com/doc/v3/merchant/4012587166.md#2.2.1%E3%80%81%E5%95%86%E6%88%B7%E5%88%9B%E5%BB%BA%E6%94%AF%E4%BB%98%E5%88%86%E8%AE%A2%E5%8D%95) |
| 测试白名单 | 上线前服务仅对白名单微信号开放，非白名单用户调用会提示“暂无法使用此服务”。需在商户平台配置微信白名单。 | [测试微信号配置指引](https://pay.weixin.qq.com/doc/v3/merchant/4012587248.md) |
| 服务商授权 | 服务商模式下，需确认特约商户已在商户平台【产品中心 -\> 我的授权产品】中手动点击“授权”支付分。 | [常见问题指引](https://pay.weixin.qq.com/doc/v3/partner/4013757943.md#Q%EF%BC%9A%E8%B0%83%E7%94%A8%E5%88%9B%E5%BB%BA%E6%94%AF%E4%BB%98%E5%88%86%E8%AE%A2%E5%8D%95%E6%8A%A5%E9%94%99%EF%BC%9A%E2%80%9C%E8%AF%B7%E6%A3%80%E6%9F%A5sub_mchid%E6%98%AF%E5%90%A6%E6%8E%88%E6%9D%83mchid%E5%BE%AE%E4%BF%A1%E6%94%AF%E4%BB%98%E5%88%86%E4%BA%A7%E5%93%81%E6%9D%83%E9%99%90%E2%80%9D%EF%BC%9F) |

## 二、上线后（运营阶段）新增风险

| 检查项 | 详细描述与解决方案 | 参考文档 |
| :-- | :-- | :-- |
| 增量配置遗漏 | 商户新增了商户号或更换了小程序appid，未同步给微信运营在原有service\_id下进行增量绑定。 | [新增配置商户号或appid指引](https://pay.weixin.qq.com/doc/v3/merchant/4012587233.md) |
| 风控/管控状态 | 商户号因违规、投诉率过高等原因被限制支付功能。 | [商户自查违规记录及申诉指引](https://kf.qq.com/faq/240129iuEZBJ240129meAR3u.html) |
