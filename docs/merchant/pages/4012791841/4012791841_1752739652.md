## 1、申请H5支付产品权限

若商户已有商户号(如没有请参考[微信支付商户入驻详细流程](https://kf.qq.com/faq/180910eUfeyu180910BFZNJN.html)指引先申请商户号)，且需要开通H5支付，可通过【[商户平台](https://pay.wechatpay.cn/index.php/core/home/login) -> 产品中心 -> H5支付 -> 申请开通 -> 填写支付域名、支付域名ICP备案截图及经营场所简介等信息 -> 提交申请】操作，待审核(7个工作日内)通过后将开通H5支付权限。

|     |     |
| --- | --- |
| 步骤一：登录商户平台，点击“产品中心” | 步骤二：点击“H5支付” |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/e92eb74f8904c0573736bbc03347a146.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/504872ae0f8bae0a08a1f264b0c8fe02.png) |
| 步骤三：点击“申请开通” | 步骤四：填写支付域名、支付域名ICP备案截图，经营场所简介等信息（详细参考下方填写说明），<br>阅读协议，确认无误后，勾选我已阅读，点击“提交申请” |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/0d90289032722d380955bd4f8ac4405d.png) | ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/ec0b37917e2bd051a94519ffb737ea70.png)<br>![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/6f6f2f34aa51d20fcee8412f45695b23.png) |
| 填写说明 |
| 支付域名 | 1、支付域名是指：商户向微信支付发起H5支付请求的域名<br>2、请填写完整的支付域名，一级域名必须能在公网可访问并可见其具体的经营内容，[《申请接入微信支付的商户H5页面内容参考》](https://kf.qq.com/faq/211124JbyEFj211124aeY77F.html)。<br>3、如果添加多个域名，请点击「+」，最多添加5个域名<br>4、如需添加多个域名，请根据页面提示分别填写每个域名对应的相关信息 |
| 支付域名ICP备案截图 | 支付域名对应的ICP备案截图。提供该链接页面([查询ICP备案](https://beian.miit.gov.cn/#/Integrated/recordQuery))查询对应支付域名ICP备案结果的截图。 |
| 经营场所简介 | 请描述经营场所里售卖的商品或者服务的所属行业、服务对象、售卖商品或者服务的简介，以及单价区间；<br>如果需要账号和密码才能访问，请在此填入。（请输入不超过500字符的内容） |
| 经营场所的一级域名是否与支付域名一致 | 选择【是】或者【否】，<br>如选择【否】，还需要说明：经营场所的一级域名与支付域名不一致的原因；以及提供“经营场所的一级域名” |
| 经营场景截图-域名首页 | 请上传支付域名或者经营场所域名的一级域名首页截图，需包含链接，[截图示例指引](https://kf.qq.com/faq/211124JbyEFj211124aeY77F.html) |
| 经营场景截图-域名经营场景页 | 请上传支付域名或者经营场所域名的商品或者服务的截图，包括链接、商品名称、商品价格与介绍，[截图示例指引](https://kf.qq.com/faq/211124JbyEFj211124aeY77F.html) |
| 经营场景截图-电商平台的店铺页（选填） | 若域名的经营性质为电商平台，请上传支付域名或者经营场所域名的店铺页和店铺资质证照页，包括：店铺名称、商品信息和店铺经营资质证明，[截图示例指引](https://kf.qq.com/faq/211124JbyEFj211124aeY77F.html) |
| 支付域名的主体或者经营场所域名的主体是否是下述信息 | 选择主体【是】或【不是】<br>ICP主办单位和商户号主体的异主体合作关系，目前只支持以下3种范围：<br>①同集团关系，需上传天眼查等公网可查的关系截图证明；<br>②品牌与经销商关系，需上传能体现双方合作方式和合作内容的证明，且域名主办单位主体的营业执照或其他资质能证明自己具备与合作内容相关的经营资质；<br>③SaaS服务，需提供能双方合作协议以及域名主办单位的经验资质。<br>如选择【不是】，需要提供：<br>【网站使用授权函】：请上传网站使用授权函，至少上传1张，最多可以上传10张图片；[授权函模版](https://kf.qq.com/faq/180315EZjIfe180315JFFVVr.html)。<br>【域名主办单位的经营资质以及与申请商户的合作协议】：至少上传2张，最多可以上传10张图片；合作协议中的敏感信息可以打码。 |
| 其他材料或者特殊资质 | 若网站经营内容涉及特殊资质的需补充相应资质或者授权证明文件。最多可上传5张；<br>如《网络文化经营许可证》、《互联网药品交易服务证》、《拍卖经营批准证书》、运营商授权证明文件、游戏版号证明、[网信办已备案的联盟链区块链技术的证明](https://bcbeian.ifcert.cn/index)等； |
