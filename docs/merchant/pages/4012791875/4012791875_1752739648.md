## 1、已有商户号申请Native支付产品权限

若商户已有商户号(未开通Native支付权限)，需新增Native支付权限，可通过【[商户平台](https://pay.wechatpay.cn/index.php/core/home/login) -> 产品中心 -> Native支付 -> 申请开通 】，待审核(7个工作日内)通过后将开通Native支付权限。

## 2、申请新商户号和Native支付产品权限

若商户没有商户号，请参考[微信支付商户入驻详细流程](https://kf.qq.com/faq/180910eUfeyu180910BFZNJN.html)指引申请商户号([入驻支持的主体类型](https://kf.qq.com/faq/180910IBZVnQ180910naQ77b.html))，申请商户号的同时可同步申请Native支付权限。

当在“填写经营与行业信息”这一步骤时，Native支付需要选择PC网站经营场景(其他场景可按需选择)，并填写网站授权函，PC网站对应的服务号或公众号APPID及PC网站域名。

| 申请过程中需特别注意的步骤 | 字段填写说明 |
| :-- | :-- |
| ![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/a96fd265e5755ae1e31ea4919003865c.png) | 商户简称 | 1、 不支持单纯以人名来命名，若为个体户经营，可用“个体户+经营者名称”或“经营者名称+业务”命名，如“个体户张三”或“张三餐饮店”；<br>2、不支持无实际意义的文案，如“XX特约商户”、“800”、“XX客服电话XXX”；<br>3、填写的商户简称会在交易场景中显示，参考：[入驻填写的商户简称展示方式](https://kf.qq.com/faq/161226RbUVvU161226JjY3M7.html) |
| 客户电话 | 1、请填写真实有效的客服电话，将在交易记录中向买家展示，提供咨询服务。<br>2、请确保电话畅通，以便入驻后微信侧回拨确认。 |
| 所属行业 | 所属行业请参考：[入驻结算规则、行业属性及特殊资质](https://kf.qq.com/faq/220228IJb2UV220228uEjU3Q.html)。 |
| 网站授权函 | 若网站域名备案主体与申请主体不同，请参照模板打印并上传：<br>1、[网站授权函](https://kf.qq.com/faq/180315EZjIfe180315JFFVVr.html)的信息全部需打印，不支持手写商户信息；<br>2、扫描件或照片大小不得超过2M，所有材料均需清晰、且盖有商户号及授权主体公章。 |
| PC网站对应的服务号或公众号APPID（选填） | 请填写已认证的服务号或公众号APPID，需是已认证的服务号、政府或媒体类型的公众号。要求申请主体一致，且服务号或公众号需要有内容。<br>1、商户如未申请公众号的appid，可参考指引注册申请：[注册公众平台账号](https://kf.qq.com/product/weixinmp.html#hid=87)。<br>2、商户如已申请公众号的appid，可登录[公众平台](https://mp.weixin.qq.com/)，参考指引[查看APPID账号](https://pay.weixin.qq.com/doc/v3/merchant/4013287246.md#2.1.1%E3%80%81%E6%9C%8D%E5%8A%A1%E5%8F%B7%2F%E5%85%AC%E4%BC%97%E5%8F%B7)。<br>注意：入驻时填写的appid，系统会自动发起商户号与该APPID的授权绑定，商户参考指引[确认授权绑定商户号](https://pay.weixin.qq.com/doc/v3/merchant/4013287246.md#2.3%E3%80%81%E7%99%BB%E5%BD%95%E5%AF%B9%E5%BA%94%E5%B9%B3%E5%8F%B0%E7%A1%AE%E8%AE%A4%E6%8E%88%E6%9D%83%E7%BB%91%E5%AE%9A%E7%9A%84%E5%95%86%E6%88%B7%E5%8F%B7)。 |
| PC网站域名 | 若为网站售卖、智能终端等场景，可填写官网链接（格式需以http://或https://开头）<br>网站域名需ICP备案，若备案主体与申请主体不同，请上传加盖公章的[网站授权函](https://wx.gtimg.com/mch/img/icp.doc)。 |
