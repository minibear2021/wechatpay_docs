## 功能介绍

商户API证书，是指由商户申请的，用来证实商户身份的证书。API证书由证书授权机构Certificate Authority（简称CA）颁发。证书中包含商户的商户号、公司名称、公钥等信息。请妥善保管API证书文件，防止泄露。

注意：

商户API证书的有效期为5年，自申请成功之日起计算。在有效期内无需定期更新，但需注意证书到期前需提前更换。

## 使用场景

APIv2 中，调用微信支付安全级别较高的接口（如：退款、企业红包、企业付款）

APIv3 中，调用微信支付所有接口的签名都需要用到商户API证书。

商户API证书具体使用说明可参见[开发必要参数说明](https://pay.weixin.qq.com/doc/v3/merchant/4013070756.md)。

## 常见问题

### Q：如何查看商户API证书序列号？

A：

1、登录[商户平台](https://pay.weixin.qq.com/)【账户中心 -\> API安全 -> 商户API证书 -> 管理证书】，可查看商户API证书序列号。

2、商户API证书和微信支付平台证书均可以使用第三方的证书[解析工具](https://myssl.com/cert_decode.html)，查看证书内容。

3、使用openssl命令行工具查看证书序列号。

```
$ openssl x509 -in 1900009191_20180326_cert.pem -noout -serial
serial=1DDE55AD98ED71D6EDD4A4A16996DE7B47773A8C
```

### Q：如何获取商户API证书私钥？

A：商户申请商户API证书时，证书工具会生成商户私钥，并保存在本地证书文件夹的文件 `apiclient_key.pem` 中。私钥也可通过工具从商户的p12证书中导出。请妥善保管好商户证书文件。

提示

请勿将私钥文件暴露在公共场合，如上传至 GitHub 或将其写入客户端代码等。

生成后的证书文件为压缩包，解压后有4个文件，证书p12格式，证书pem格式，证书私钥pem格式，说明。

### Q：如何查看商户API证书使用量？

A：可登录[商户平台](https://pay.weixin.qq.com/)【账户中心 -\> API安全 -> 商户API证书 -> 管理证书】可查看商户API证书昨日使用量。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/e32d2b820ee81a0bb9bd7606ddc94e6b.png)

### Q：在商户API证书即将到期时，如果旧证书未申请作废，是否可以在证书到期后直接使用新申请的证书？

A：商户API证书到期时就会自动作废，用过期证书无法调用v3接口，所以请务必在到期前完成证书更换。申请新证书不影响旧证书继续使用，微信支持最多3份API证书同时生效，使用其中任意一份有效证书均可。并且只要不超过数量限制，是否作废旧证书都可以随时申请新证书。

### Q：之前使用的v2版本接口已经申请过商户API证书，现在接入v3版本接口是否需要重新申请商户API证书？

A：不需要。

### Q：商户API证书申请后是立即生效么？是否可以立即替换旧证书使用？

A：是的，可以立即替换。

### Q：商户号只有一个证书怎么作废掉？

A：可以申请一个新证书，然后把旧证书作废。

### Q：商户API证书到期，申请新商户API证书后，需要更换几个参数？

A：

1、APIv3接口需要更换代码中的【商户API证书私钥】和【商户API证书序列号】。

2、APIv2接口需要更换请求接口时携带的apiclient\_cert.p12商户API证书文件，详细参考[商户API证书](https://pay.weixin.qq.com/doc/v2/merchant/4011985891.md#4%E3%80%81API%E8%AF%81%E4%B9%A6)。

### Q：APIv2接口与APIv3接口是否可以使用同一个商户API证书的公钥/私钥？

A：支持。
