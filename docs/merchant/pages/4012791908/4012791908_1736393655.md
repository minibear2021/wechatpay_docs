## 下载接口说明

微信支付按天提供商户各账户的资金流水账单文件，商户可以通过该接口获取账单文件的下载地址。账单文件详细记录了账户资金操作的相关信息，包括业务单号、收支金额及记账时间等，以便商户进行核对与确认。详细介绍参考：[下载账单-产品介绍](https://pay.weixin.qq.com/doc/v3/merchant/4013071215.md)。

注意：

- 资金账单中的数据反映的是商户微信账户资金变动情况；

- 当日账单将在次日上午9点开始生成，建议商户在次日上午10点以后获取；

- 资金账单中所有涉及金额的字段均以“元”为单位。

- 以商户号维度频率限制为3QPS。


## 文件格式说明

账单文件主要由明细数据和汇总数据两大部分构成，每部分均包含一行表头以及多行详细数据。

明细数据的每一行都代表一笔具体的资金操作。为防止数据在Excel中被自动转换为科学计数法，每项数据前均添加了字符 `` ` ``。若需汇总计算金额等数据，可以批量移除该字符。

## 接口说明

支持商户：【普通商户】

请求方式：【GET】 `/v3/bill/fundflowbill`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| Authorization | string | true | 请参考 [签名认证](https://pay.weixin.qq.com/doc/v3/merchant/4012365342.md#1.1-%E8%AF%B7%E6%B1%82%E5%BE%AE%E4%BF%A1%E6%94%AF%E4%BB%98%E6%8E%A5%E5%8F%A3) 生成认证信息 |
| Accept | string | true | 请设置为 `application/json` |
### Query 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| bill_date | string(10) | true | 【账单日期】 账单日期，格式yyyy-MM-DD，仅支持三个月内的账单下载申请。 |
| account_type | string | false | 【资金账户类型】 资金账户类型，不填默认是BASIC<br>可选取值<br><ul><li>`BASIC`:  基本账户</li><li>`OPERATION`:  运营账户</li><li>`FEES`:  手续费账户</li></ul> |
| tar_type | string | false | 【压缩类型】 压缩类型，不填则以不压缩的方式返回账单文件流。<br>可选取值： `GZIP`: 下载账单时返回.gzip格式的压缩文件流 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| hash_type | string | true | 【哈希类型】 哈希类型，固定为SHA1。 |
| hash_value | string(1024) | true | 【哈希值】账单文件的SHA1摘要值，用于商户侧校验文件的一致性。 |
| download_url | string(2048) | true | 【下载地址】 供下一步请求账单文件的下载地址，该地址5min内有效。参考 [下载账单](https://pay.weixin.qq.com/doc/v3/merchant/4013071238.md) |

| 状态码 | 错误码 | 描述 | 解决方案 |
| --- | --- | --- | --- |
| 400 | PARAM\_ERROR | 参数错误 | 请根据错误提示正确传入参数 |
| 400 | INVALID\_REQUEST | HTTP 请求不符合微信支付 APIv3 接口规则 | 请参阅 [接口规则](https://pay.weixin.qq.com/doc/v3/merchant/4012081709.md) |

### 业务错误码

| 状态码 | 错误码 | 描述 | 解决方案 |
| --- | --- | --- | --- |
| 400 | INVALID\_REQUEST | 参数错误 | 请根据接口返回的错误描述检查参数，参数需按文档要求进行填写 |
| 400 | NO\_STATEMENT\_EXIST | 账单文件不存在 | 请检查当前商户号的指定账户是否在指定日期有资金变更。 |
