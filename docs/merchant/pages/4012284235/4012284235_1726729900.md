## 接口参数

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| mchid | string(32) | true | 商户号。 |
| openid | string(32) | true | 用户在商户对应AppID下的唯一标识。 |
| plate_number | string(32) | true | 待开通车牌。 |
| plate_color | string(32) | true | 待开通车牌的颜色<br>BLUE：蓝色<br>GREEN：绿色<br>YELLOW：黄色<br>BLACK：黑色<br>WHITE：白色<br>LIMEGREEN：黄绿色 |
| trade_scene | string(32) | false | 开通场景信息，目前只支持PARKING |
