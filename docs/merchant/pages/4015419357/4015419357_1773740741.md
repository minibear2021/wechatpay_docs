# 一、背景

微信支付APIv3支持平台证书和微信支付公钥两种模式构造签名验签，新申请的商户号默认为微信支付公钥模式。商户可通过自主操作，切换至平台证书模式。

请注意，平台证书有效期为5年，每5年商户需主动更换一次，若你选择切换至平台证书模式，要确保知悉证书过期风险。

# 二、切换步骤

## 1、超级管理员登录商户平台-API安全-微信支付公钥

点击下载公钥后，入口出现

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/1c14fad381dcb346063f037039adbdad.png)

## 2、按导航步骤完成，即可切换到平台证书模式

### 第一步：使用知悉

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/d9f16b9b00746c692f66172a246b48e7.png)

### 第二步：请确认你的系统已经没有在使用微信支付公钥

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/a42675b26f0390913ded0bc154bb82ac.png)

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/9fb5e6fd095df41d19dbc40ba739c27d.png)

### 第三步：回答证书和微信支付公钥相关问题，确保你已经知悉证书和公钥的使用方式

两道题均为多选，需要回答正确，才可以进行下一步

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/b5d72e9dcd97345d830249625b20b68d.png)

### 第四步：确认申请信息，配置安全联系人，填写贵司系统情况

未曾使用过平台证书的商户，无需选择，只能生成新平台证书。曾使用过平台证书的商户，需要先选择本次申请哪种平台证书。

（1）曾使用的平台证书已经过期，默认选择生成新的平台证书

系统将默认选择生成新的平台证书，无法选择过期已有平台证书。如有需要，可点击查看已过期证书信息。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/9118fd84156b3b42acc0c96f8f1efa97.png)

（2）曾使用的平台证书仍在有效期内，可选择使用已有平台证书或生成新平台证书

若商户计划使用已有平台证书，请点击查看证书信息，重点关注有效期，评估在过期之前能否平滑切换到新证书。

若难以及时平滑切换，建议直接选择生成新平台证书，微信支付会为商户生成新平台证书，并将已有的平台证书设置为立即过期失效。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/4b5e64bfe8d64d679c333fcb3b8b44b0.png)

完成上述选择后，还需要填写系统信息并配置安全联系人。请如实选择贵司的系统情况，以便我司知悉贵司的安全风险。

注意，该步骤需要配置安全联系人，通过图中蓝色链接可以跳转。

安全联系人具有的权限有：操作/修改API证书、密钥、平台证书、微信支付公钥，同时接收商户的安全通知。

![](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/img/55a9cfd2d9893dd1a6c74a30e3a2a899.png)
