## 1、资金账单文件内容

明细内容：文件内包含商户账户内资金操作相关的业务单号、收支金额、记账时间等信息。

资金账单的明细数据表头、汇总数据表头所包含的字段具体如下：

| 明细数据表头字段 | 汇总数据表头字段 | 账单文件样例 |
| :-- | :-- | :-- |
| 记账时间,微信支付业务单号,资金流水单号,业务名称,业务类型,收支类型,收支金额(元),账户结余(元),资金变更提交申请人,备注,业务凭证号 | 资金流水总笔数,收入笔数,收入金额,支出笔数,支出金额 | [资金账单.xlsx](https://gtimg.wechatpay.cn/resource/xres/mmpaydoc/static/attachment/a84c90eae7c6eb0da29662d4594c4dd3/资金账单.xlsx) |
| 注意事项：<br>- 文件内每个字段都使用英文逗号（,）间隔。明细数据和汇总数据每个字段前会增加1个（\`）字符（用于避免获取的内容被excel展示为科学计数法的格式、丢失数据细节）。<br>  <br>- 文件样例仅为展示示例，实际API下载的是数据流。 |

## 3、特殊字符转义

资金账单会对字段取值中含有的\\1字符、\\r字符、\\n字符做替换处理，其中\\1会替换为空格，\\r字符和\\n字符会分别替换为\\\r字符,\\\n字符

针对明细数据中的备注、业务凭证号这两个商户自定义字段，账单在处理过程中对其进行了特殊字符的转义，具体规则包括：

| 原始特殊字符<br>Unicode | 含义 | 转义后<br>Unicode | 含义 |
| :-- | :-- | :-- | :-- |
| `\u000a` | 换行符(LF) | `\u005c\u006e` | \\n |
| `\u000d` | 回车符(CR) | `\u005c\u0072` | \\r |
| `\u0001` | 控制字符SOH(\\001) | `\u0020` | 空格 |
