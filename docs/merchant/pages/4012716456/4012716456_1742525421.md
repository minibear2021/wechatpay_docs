商户可以 指定微信转账单号通过该接口申请 商家转账用户确认模式转账单据 对应的电子回单。

微信支付会在校验满足回单申请条件后受理回单的申请，商户后续可以通过电子回单查询接口查询回单处理进度。

申请回单前，请先确认是否满足回单申请的条件，不满足回单申请条件时，多次重试申请无意义：

1. 仅支持状态为SUCCESS的转账单据申请电子回单.

2. 仅支持传入了收款用户姓名的转账单据申请电子回单.

3. 仅支持六个月内的转账单据申请电子回单.

回单申请完成后的有效期为90天，过期后需要重新申请。受理成功不表示回单一定能成功生成，以查单结果返回为准。


## 接口说明

支持商户：【普通商户】

请求方式：【POST】 `/v3/fund-app/mch-transfer/elecsign/transfer-bill-no`

请求域名：【主域名】https://api.mch.weixin.qq.com 使用该域名将访问就近的接入点

　　　　　【备域名】https://api2.mch.weixin.qq.com 使用该域名将访问异地的接入点，指引[点击查看](https://pay.weixin.qq.com/doc/v3/merchant/4012075113.md)

## 请求参数

### Header 参数
| 参数名 | 类型 | 必填 | 描述 |

### Body 参数
| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| transfer_bill_no | string(64) | true | 【微信转账单号】 转账单据对应的微信支付单号 |
## 应答参数

200 OK

| 参数名 | 类型 | 必填 | 描述 |
| --- |  --- | --- | --- |
| state | string | true | 【申请单状态】 GENERATING：生成中，FINISHED: 已完成，FAILED: 已失败<br>可选取值<br><ul><li>`GENERATING`:  表示当前电子回单已受理成功并在处理中</li><li>`FINISHED`:  表示当前电子回单已处理完成</li><li>`FAILED`:  表示当前电子回单生成失败，失败原因字段会返回具体的失败原因</li></ul> |
| create_time | string(32) | true | 【申请单创建时间】 申请单的创建时间，按照使用rfc3339所定义的格式，格式为yyyy-MM-DDThh:mm:ss+TIMEZONE |
