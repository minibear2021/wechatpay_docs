在使用商家转账时，必须设置请求来源IP 。仅设定IP可调用商家转账API。未设置请求接口将报错“此IP地址不允许调用接口，请按开发指引设置”。该项设置可避免商户API密钥泄露后，被陌生IP发起调用盗走资金。同时微信支付提供了免费的系统安全检测服务，请求商家转账 API 的服务器 IP建议先在安全医生中通过安全诊断。

IP地址支持超级管理员和[安全联系人配置](https://pay.weixin.qq.com/doc/v3/merchant/4012076563.md)

设置路径：[商户平台](https://pay.wechatpay.cn/index.php/public) > 产品中心 > 运营工具 > 商家转账 \> 前往功能 >安全能力或者直接进入[系统信息管理](https://pay.weixin.qq.com/xdc/mchsecipweb/index.html#/)进行配置。
